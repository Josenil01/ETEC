Aula 16: Encantamentos Reutilizáveis

Não repita o seu código. Aprenda a criar funções poderosas com parâmetros e retornos para tornar os seus scripts profissionais e organizados.

📚 Resumo Teórico

Uma Função é um bloco de código que só é executado quando é chamado. Pense nela como um "feitiço" guardado num grimório: você escreve a lógica uma vez e usa-a quantas vezes quiser.

  • Definição (def): É onde criamos a função.
  • Parâmetros: São as informações que enviamos para dentro da função (os ingredientes).
  • Retorno (return): É o resultado que a função nos devolve (a poção pronta).

Usar funções permite o DRY (Don't Repeat Yourself), o mantra sagrado dos programadores.

1. Anatomia de uma Função

Para criar uma função em Python, usamos a palavra-chave def, seguida do nome e parênteses. A indentação define o corpo da função.

Estrutura Visual
def nome_da_funcao(parametro):
    # lógica aqui
    return resultado

🔹 Nome: Deve ser claro e usar snake_case.

🔹 Parênteses: Onde declaramos as variáveis de entrada.

🔹 Return: Termina a função e envia o valor para quem a chamou.

3. Laboratório: O Sistema de Cura Automático

Este programa integra Dicionários, Listas, If/Else e Funções para gerir a saúde de uma equipa.

cura_equipa.py 
def aplicar_pocao(heroi, valor_cura):
    heroi["vida"] += valor_cura
    if heroi["vida"] > 100:
        heroi["vida"] = 100
    print(f"{heroi['nome']} recuperou saúde para {heroi['vida']} HP!")

# Dados do jogo
equipa = [
    {"nome": "Aragorn", "vida": 45},
    {"nome": "Gimli", "vida": 30}
]

cura_input = int(input("Poder da poção encontrada: "))

# Usando a função num laço
for aventureiro in equipa:
    if aventureiro["vida"] < 50:
        aplicar_pocao(aventureiro, cura_input)

4. Erros Arcanos: O Labirinto das Funções

Problemas de Escopo (Scope)

Variáveis criadas dentro de uma função são locais. Se tentares usá-las fora, o código vai "crashar".

def soma(a, b):
  resultado = a + b
print(resultado) # ERRO: resultado não existe aqui!

Confusão Print vs Return

O print() apenas mostra na tela. O return envia o valor para ser guardado ou processado depois.

def dobro(n):
  print(n * 2) # Apenas exibe
x = dobro(5) # x ficará VAZIO (None)

Esquecer os Parênteses

Para executar uma função, precisas dos parênteses (). Sem eles, estás apenas a referenciar o objeto função.

def magia(): print("Fogo!")
magia # Não faz nada
magia() # AGORA SIM executa!

Indentação no Return

O return finaliza a função. Se o colocares dentro de um laço por engano, o laço irá parar na primeira volta.

def erro():
  for i in range(5):
    return i # Vai parar no 0 e sair!
