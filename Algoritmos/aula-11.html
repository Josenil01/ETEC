1. Uma Nova Sintaxe, a Mesma Lógica

Programar em Python é como escrever em inglês simplificado. A linguagem foi desenhada para ser lida facilmente por humanos. No entanto, ela exige disciplina: a indentação (espaços no início da linha) não é apenas estética, é o que define a estrutura do teu código.

O que desaparece?

  • As chavetas { }
  • O ponto e vírgula ;
  • Palavras como inicio e fim

O que aparece?

  • Os dois pontos :
  • Identação obrigatória
  • Tipagem dinâmica (não precisas declarar tipos)

5. Laboratório: Primeiras Runas

01 Ficha de Atributos (Entrada e Formatação)

O Primeiro Script!

Digita este código para entenderes como as variáveis funcionam sem declaração de tipo.

ficha_heroi.py 
nome = input("Qual o nome do teu aventureiro? ")
nivel = int(input("Qual o nível inicial? "))
saude_total = 100 + (nivel * 10)

print(f"FICHA DE PERSONAGEM: {nome}")
print(f"NÍVEL: {nivel} | SAÚDE: {saude_total} HP")

02 A Alquimia dos Números (Casting e Aritmética)

calculadora_mana.py 
mana_atual = float(input("Mana atual: "))
regeneracao = float(input("Regeneração por turno: "))
mana_apos_5_turnos = mana_atual + (regeneracao * 5)

print(f"Em 5 turnos, terás {mana_apos_5_turnos:.2f} de mana total.")

03 O Espelho da Verdade (Operadores Lógicos)

portal_logico.py 
tem_chave = True
nivel_heroi = 12
pode_entrar = tem_chave and nivel_heroi >= 10

print(f"O aventureiro pode atravessar o portal? {pode_entrar}")
