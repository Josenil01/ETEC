2. O Primeiro Encantamento em Java

01. Olá Mundo e Ficha de Personagem

Repara que no Java tudo deve estar dentro de uma Classe. E o nome do ficheiro DEVE ser o mesmo da classe.

Ponto Crucial!

Salva este ficheiro como Principal.java no teu computador.

Principal.java 
// Todo o programa Java começa numa classe
public class Principal {

    // O método main é o portal de entrada do programa
    public static void main(String[] args) {
        
        // DECLARAÇÃO DE VARIÁVEIS (Tipagem Forte)
        String nomeHeroi = "Aragorn";
        int nivel = 10;
        double moedasOuro = 150.75;
        boolean temArmadura = true;

        // SAÍDA DE DADOS (System.out.println)
        System.out.println("Bem-vindo ao reino do Java!");
        System.out.println("Heroi: " + nomeHeroi);
        System.out.println("Nivel: " + nivel);
        System.out.println("Ouro: " + moedasOuro + " moedas");
    }
}

Decifrando os Selos Reais: Public Class e Static

Para quem vem do Portugol ou do Python, o início de um ficheiro Java parece um código encriptado. Vamos fazer a ponte entre o que já sabes e esta nova estrutura:

🛡️ Public Class

No Portugol, começavas com programa { }. No Java, a Class é o equivalente. É a "muralha" que define o teu domínio.

O public significa que este código é acessível por todo o reino (outros ficheiros), permitindo que o sistema encontre o teu herói.

⚡ Public Static Void Main

Lembras-te do funcao inicio()? O main é exatamente isso: o gatilho de partida.

O selo static é o mais especial: ele garante que este "portal de entrada" existe logo que o programa é carregado, sem que tenhas de construir um castelo (objeto) primeiro. É a fundação mágica de tudo.

Paralelo Mestre: Se o código fosse uma cidade, a class seriam os limites do município e o static main seria o Portão Principal que está sempre aberto para o sistema operacional entrar.

3. Erros Arcanos: Choque de Tipos

O Ponto e Vírgula (;)

No Java, o ponto e vírgula é OBRIGATÓRIO no final de cada comando. Esquecer um causa erro de compilação imediato.

Case Sensitive Extremo

String (com S maiúsculo) é diferente de string. O Java não perdoa diferenças entre maiúsculas e minúsculas.
