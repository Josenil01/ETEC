Módulo 1 Laços de Repetição Avançados

Aula 09: A Marcha dos Soldados

Mantenha o ritmo e o controle. Aprenda a usar o laço "Para" para repetições com contagem determinada e precisão absoluta.

1. Quando usar o "Para"?

Diferente do enquanto, que é movido por uma condição que pode mudar a qualquer momento, o laço Para (For) é usado quando sabemos exatamente **quantas vezes** queremos repetir algo. É a estrutura perfeita para contar passos, processar listas ou organizar fileiras de soldados.

"O 'Enquanto' espera por um sinal. O 'Para' marcha até o destino final."

2. A Trindade do Controle

A estrutura do para condensa três passos fundamentais numa única linha:

Inicialização

Onde a contagem começa. Ex: i = 1

Condição

Até onde o laço deve ir. Ex: i <= 10

Incremento

Como a contagem avança. Ex: i++ ou i = i + 1

3. Laboratório de Marcha

01 A Fileira de Soldados (Laço Para Básico)

Precisão Militar!

Observe como a variável i controla o ritmo da marcha.

marcha_soldados.por 
programa {
  funcao inicio() {
    // i é a variável de controle (nosso contador)
    para (inteiro i = 1; i <= 10; i++) {
        escreva("Soldado nº ", i, " apresente-se!\n")
    }
    escreva("Tropa formada e pronta para o combate!")
  }
}

02 Contagem Regressiva Explosiva (Para + IF)

bomba_alquimica.por 
programa {
  funcao inicio() {
    escreva("Bomba alquímica ativada!\n")
    
    // i-- faz a contagem diminuir (decrementar)
    para (inteiro i = 10; i >= 1; i--) {
        escreva(i, "...\n")
        
        se (i == 5) {
            escreva("AVISO: Metade do tempo restante!\n")
        }
    }
    escreva("KABOOM! 🔥")
  }
}

03 Distribuição de Rações (Para + Switch Case)

Logística de Guerra!

Cada soldado escolhe o seu suprimento através de um menu repetido.

logistica_exercito.por 
programa {
  funcao inicio() {
    inteiro tipo_racao
    
    para (inteiro s = 1; s <= 3; s++) {
        escreva("\nSoldado ", s, ", escolhe ração (1-Carne / 2-Pão): ")
        leia(tipo_racao)
        
        escolha (tipo_racao) {
            caso 1:
                escreva("Ração de Proteína entregue.\n")
                pare
            caso 2:
                escreva("Ração de Carboidrato entregue.\n")
                pare
            outro_caso:
                escreva("Soldado ficou com fome por escolha errada!\n")
        }
    }
  }
}

04 Contagem de Tesouros (Para + Acumulador)

Ouro Total!

Usamos o laço para somar o conteúdo de vários baús encontrados.

tesouro_acumulado.por 
programa {
  funcao inicio() {
    inteiro moedas_no_bau, ouro_total = 0
    
    para (inteiro bau = 1; bau <= 5; bau++) {
        escreva("Quantas moedas no baú nº ", bau, "? ")
        leia(moedas_no_bau)
        
        // Acumulador: soma o valor antigo com o novo
        ouro_total = ouro_total + moedas_no_bau
    }
    
    escreva("\nInventário finalizado. Total de Ouro: ", ouro_total)
  }
}

05 Recrutamento de Elite (Para + Se + Operadores)

Seleção Arcaica!

Avaliamos 5 candidatos e apenas aqueles com força > 70 são recrutados.

selecao_elite.por 
programa {
  funcao inicio() {
    inteiro força, recrutas = 0
    
    para (inteiro c = 1; c <= 5; c++) {
        escreva("Força do candidato ", c, ": ")
        leia(força)
        
        se (força > 70) {
            escreva("Candidato Aprovado! ✅\n")
            recrutas++
        } senao {
            escreva("Candidato Reprovado. ❌\n")
        }
    }
    
    escreva("\nRecrutamento encerrado. Soldados de elite obtidos: ", recrutas)
  }
}
