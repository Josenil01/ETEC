1. A Dança dos Ciclos

Em Python, tal como em Portugol, temos duas formas principais de repetir código. A grande diferença é a **simplicidade** com que o Python gere estas repetições.

while (Enquanto)

Repete enquanto uma condição for verdadeira. Ideal para quando **não sabemos** quando o utilizador vai parar.

for (Para)

Repete sobre uma sequência ou intervalo. Ideal para quando **sabemos exatamente** o número de voltas.

2. O Loop Infinito Controlado (While)

menu_eterno.py 
# Exemplo de menu que só fecha quando o utilizador quer
opcao = 0

while opcao != 3:
    print("\n--- TAVERNA DO CÓDIGO ---")
    print("1. Comprar Poção")
    print("2. Ver Inventário")
    print("3. Sair da Taverna")
    
    opcao = int(input("Escolha uma ação: "))
    
    if opcao == 1:
        print("Poção de Mana adicionada! -20 moedas.")
    elif opcao == 2:
        print("O seu inventário está vazio... por agora.")
    elif opcao == 3:
        print("Até à próxima, aventureiro!")
    else:
        print("Opção inválida! Tenta de novo.")

3. A Marcha Precisa (For + Range)

Em Python, o for trabalha muito com a função range(), que gera uma sequência de números para nós.

treino_arqueiro.py 
# range(1, 6) gera: 1, 2, 3, 4, 5 (o último nunca entra)
print("Iniciando treino de pontaria...")

for flecha in range(1, 6):
    print(f"Flecha nº {flecha} disparada!")
    acertou = input("Acertou no alvo? (s/n): ")
    
    if acertou == 's':
        print("Bela pontaria!")
    else:
        print("Mais foco na próxima.")

print("Treino concluído. Aljava vazia.")
Anatomia do Range: range(inicio, fim, passo).
Exemplo: range(0, 10, 2) gera 0, 2, 4, 6, 8.

4. Laboratório: O Somatório de Tesouros

Vamos integrar Entrada, Loop e Acumuladores.

tesouro_total.py 
total_ouro = 0
baus = int(input("Quantos baús encontraste na masmorra? "))

for i in range(1, baus + 1):
    moedas = int(input(f"Quantas moedas no baú {i}? "))
    total_ouro += moedas # Forma curta de total_ouro = total_ouro + moedas

print("\n--- RESULTADO DA EXPLORAÇÃO ---")
print(f"Total arrecadado: {total_ouro} moedas.")

if total_ouro > 500:
    print("És um aventureiro rico!")
else:
    print("Precisas de explorar masmorras mais perigosas.")
