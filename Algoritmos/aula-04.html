Aula 04: A Alquimia dos Números

Aprenda a misturar valores usando operadores para criar fórmulas poderosas e descobrir segredos escondidos nos dados.

1. A Arte de Combinar Valores

Na programação, uma expressão é qualquer combinação de valores e operadores que resulta num novo valor. É como uma receita de alquimia: você junta ingredientes (números ou variáveis) e aplica um processo (operador) para obter uma poção final.

2. O Caldeirão de Operadores

Aritméticos

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Resto da Divisão

Relacionais

  • == Igual a
  • != Diferente de
  • > Maior que
  • < Menor que
  • >= Maior ou Igual

Lógicos

  • e (And) Ambas verdade
  • ou (Or) Pelo menos uma
  • nao (Not) Inverte o valor

3. Laboratório de Poções (Aritmética)

Mestre Alquimista v2.0

Este código utiliza TODOS os operadores aritméticos para calcular custo, volume, bónus e frascos.

laboratorio_alquimia.por 
programa {
  funcao inicio() {
    real fogo, estrelas, custo_grama, volume
    inteiro frascos, resto

    escreva("Gramas de Essência de Fogo: ")
    leia(fogo)
    escreva("Gramas de Pó de Estrelas: ")
    leia(estrelas)
    escreva("Custo por grama: ")
    leia(custo_grama)

    real peso = fogo + estrelas
    real custo = peso * custo_grama
    volume = (peso * 2.0) - 5.0 // -5ml desperdiçado
    frascos = volume / 50 // frascos de 50ml
    resto = volume % 50

    escreva("Custo: ", custo, "\nFrascos: ", frascos, "\nSobra: ", resto, "ml")
  }
}

4. O Espelho da Verdade (Comparadores)

Comparadores relacionais servem para medir a verdade entre dois valores. O resultado de qualquer comparação é sempre um valor lógico (Verdadeiro ou Falso).

Teste de Aptidão!

Vamos verificar se um herói tem os requisitos mínimos para empunhar a lendária Excalibur.

teste_requisitos.por 
programa {
  funcao inicio() {
    inteiro forca_heroi, nivel_heroi
    inteiro REQUISITO_FORCA = 50
    inteiro REQUISITO_NIVEL = 10

    escreva("Qual a força do herói? ")
    leia(forca_heroi)
    escreva("Qual o nível do herói? ")
    leia(nivel_heroi)

    // Operadores Relacionais gerando valores lógicos
    logico tem_forca = forca_heroi >= REQUISITO_FORCA
    logico tem_nivel = nivel_heroi == REQUISITO_NIVEL
    logico nivel_superior = nivel_heroi > REQUISITO_NIVEL

    escreva("Tem força mínima? ", tem_forca, "\n")
    escreva("Nível é exatamente 10? ", tem_nivel, "\n")
    escreva("Nível é maior que o requisito? ", nivel_superior)
  }
}

5. O Oráculo de Cristais (Conectivos Lógicos)

Conectivos lógicos permitem unir várias comparações numa única resposta final. Eles são o "e", o "ou" e o "não" que governam as regras complexas do nosso código.

O Portão das Sombras!

Para abrir o portão, você precisa de uma chave mágica E um nível alto, OU ser um administrador do sistema.

oraculo_logico.por 
programa {
  funcao inicio() {
    logico possui_chave, eh_admin
    inteiro nivel_heroi

    escreva("O herói possui a chave? (verdadeiro/falso): ")
    leia(possui_chave)
    escreva("Nível do herói: ")
    leia(nivel_heroi)
    escreva("É administrador do jogo? (verdadeiro/falso): ")
    leia(eh_admin)

    // Combinação Logica complexa
    // Abre se (Tem Chave E Nível > 20) OU se for Admin
    logico pode_abrir = (possui_chave e nivel_heroi > 20) ou eh_admin

    // Operador de negação (Inverte o valor)
    logico portao_trancado = nao pode_abrir

    escreva("\nO portão se abriu? ", pode_abrir)
    escreva("\nAinda estamos trancados? ", portao_trancado)
  }
}
