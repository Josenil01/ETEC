Aula 22: Decisões Arcaicas

O herói precisa de escolher o seu caminho. Aprende a usar o if, else if e o clássico switch para ramificar a tua lógica.

📚 Resumo Teórico

No Java, as estruturas de decisão voltam a usar **chavetas `{}`** para delimitar blocos, tal como no Portugol.

  • If / Else: Usado para perguntas binárias (Verdadeiro ou Falso).
  • Else If: Diferente do Python (`elif`), aqui escrevemos as duas palavras separadas.
  • Switch-Case: Uma estrutura especializada para menus. Compara uma variável contra vários valores fixos. É mais rápido e limpo do que vários ifs encadeados.

1. A Sintaxe do Guardião

O Bloco If-Else
if (vida < 20) {
    // Corre!
} else if (vida < 50) {
    // Cuidado
} else {
    // Ataca!
}
O Menu Switch
switch (opcao) {
    case 1:
        // Acção
        break;
    default:
        // Inválido
}

2. Exemplos Práticos Comentados

01. Portão de Segurança (If/Else + Scanner)

No Java, para ler dados, precisamos da classe Scanner. Repara na importação no topo.

Portaria.java 
import java.util.Scanner;

public class Portaria {
    public static void main(String[] args) {
        Scanner leitor = new Scanner(System.in);

        System.out.print("Qual o teu nível de aventureiro? ");
        int nivel = leitor.nextInt();

        if (nivel >= 50) {
            System.out.println("Bem-vindo ao Salão dos Lendários!");
        } else if (nivel >= 10) {
            System.out.println("Podes entrar no Pátio de Treino.");
        } else {
            System.out.println("Acesso negado. Volta quando fores mais forte.");
        }
        
        leitor.close();
    }
}

02. Menu do Taberneiro (Switch-Case)

O switch é perfeito quando temos opções numeradas ou letras específicas.

Taverna.java 
import java.util.Scanner;

public class Taverna {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        System.out.println("--- MENU DA TAVERNA ---");
        System.out.println("1. Hidromel\n2. Sopa de Troll\n3. Sair");
        int escolha = input.nextInt();

        switch (escolha) {
            case 1:
                System.out.println("Aqui tens um caneco bem fresco!");
                break; 
            case 2:
                System.out.println("Cuidado, a sopa ainda se mexe...");
                break;
            case 3:
                System.out.println("Boa viagem, aventureiro!");
                break;
            default:
                System.out.println("Não percebo esse sotaque. Escolhe 1, 2 ou 3.");
        }
        input.close();
    }
}

3. Erros Arcanos: O Labirinto das Decisões

O Esquecimento do "break"

No switch, se não colocares o break, o Java continuará a executar todos os casos abaixo até encontrar um break ou o fim da estrutura.

Comparar Strings com ==

No Java, não deves usar == para comparar textos. Deves usar texto.equals("valor") para verificar o conteúdo real.

O Enigma da Esfinge
