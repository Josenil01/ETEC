O objetivo hoje é criar um simulador de RPG via terminal. O programa deve ser capaz de:

✔

Criar um herói (Dicionários e Inputs).

✔

Gerir um inventário (Listas e Loops).

✔

Sistema de combate aleatório (Módulos e Funções).

✔

Guardar progresso em disco (Tratamento de Ficheiros).

1. Planta do Código (Arquitetura)

Um projeto de mestre começa com organização. Eis como vamos dividir a nossa lógica:

Dados (Status)

  • • Hero: Dicionário {nome, lvl, hp, mana, gold}
  • • Inventory: Lista ['Espada', 'Poção']
  • • Save File: personagem.txt ou save.csv

Ações (Funções)

  • • atacar(): Gera dano baseado na força + sorte
  • • salvar_jogo(): Escreve os dados no ficheiro JSON
  • • main(): O ciclo principal do jogo (While)

2. O Enunciado da Missão

"Como arquiteto desta guilda, deves construir um sistema que sobreviva às intempéries do terminal. Um aventureiro deve poder entrar, lutar e sair com as suas riquezas preservadas."

Nesta prova final, vamos construir um jogo de RPG baseado em texto, onde o sistema solicita inicialmente ao usuário o nome do seu aventureiro para forjar sua ficha de personagem única. O herói navegará por um mundo de perigos imprevistos, enfrentando combates intensos contra criaturas geradas aleatoriamente, administrando recursos vitais no seu inventário (como poções de vida) e utilizando cálculos de dano dinâmicos. Ao final da sessão, o sistema deve permitir que toda a jornada seja imortalizada em um arquivo de persistência, garantindo que a lenda do herói sobreviva mesmo após o terminal ser fechado.

Requisitos Obrigatórios:

  • O herói deve ser um Dicionário contendo: nome, hp, força e uma lista de inventário.
  • Deves criar uma Função de ataque que use o módulo random para variar o dano.
  • O jogo deve rodar num laço While até que o herói morra ou escolha sair.
  • O herói deve poder usar uma poção do inventário (se tiver) para curar o HP.
  • Deve existir uma opção para Salvar o estado do personagem num ficheiro local.

import random
import json # Forma avançada de salvar dicionários

# --- FUNÇÕES (Aula 16) ---
def atacar(forca):
    return forca + random.randint(1, 10)

def salvar_jogo(heroi):
    try: # Tratamento de Erros (Aula 18)
        with open("savegame.json", "w") as f:
            json.dump(heroi, f)
        print("✔️ Progresso guardado no pergaminho digital!")
    except Exception as e:
        print(f"❌ Falha ao salvar: {e}")

# --- INICIALIZAÇÃO (Aula 15 e 11) ---
print("--- BEM-VINDO AO CHRONICLES OF PYTHON ---")
nome_input = input("Como te chamas, aventureiro? ")
heroi = {
    "nome": nome_input,
    "hp": 100,
    "forca": 15,
    "inventario": ["Poção de Vida"] # Lista (Aula 14)
}

# --- CICLO DE JOGO (Aula 13) ---
while heroi["hp"] > 0:
    print(f"\nHP: {heroi['hp']} | Itens: {len(heroi['inventario'])}")
    opcao = input("[1] Explorar | [2] Usar Poção | [3] Salvar e Sair: ")

    if opcao == "1":
        inimigo_hp = 30
        print("⚔️ Um Goblin apareceu!")
        while inimigo_hp > 0 and heroi["hp"] > 0:
            dano = atacar(heroi["forca"])
            inimigo_hp -= dano
            print(f"Causaste {dano} de dano!")
            if inimigo_hp > 0:
                heroi["hp"] -= 10
                print("O Goblin contra-atacou! -10 HP.")
        if heroi["hp"] > 0: print("Vitória!")

    elif opcao == "2":
        if "Poção de Vida" in heroi["inventario"]:
            heroi["hp"] = min(100, heroi["hp"] + 50)
            heroi["inventario"].remove("Poção de Vida")
            print("Saúde recuperada!")
        else:
            print("Não tens poções!")

    elif opcao == "3":
        salvar_jogo(heroi)
        break

if heroi["hp"] <= 0: print("MORRESTE. Fim da lenda.")
