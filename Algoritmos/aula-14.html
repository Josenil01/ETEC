📚 Resumo Teórico

Uma Lista é uma estrutura de dados que permite armazenar múltiplos valores em uma única variável. Diferentemente de variáveis simples, as listas são ordenadas (começam sempre no índice 0) e mutáveis (podemos modificar seus elementos).

As listas são fundamentais em programação porque permitem:

  • Armazenar coleções de dados relacionados
  • Acessar elementos por posição (índice)
  • Adicionar, remover e modificar elementos
  • Iterar sobre todos os elementos facilmente

1. O Que são Listas?

Uma lista em Python é como uma mochila com compartimentos numerados. Cada compartimento (índice) contém um valor.

Índice 0 Escudo
Índice 1 Poção
Índice 2 Mapa

2. Exemplos Práticos Comentados

Criar uma lista e acessar elementos

criar_lista.py 
# Criar uma lista com itens
inventario = ["Espada", "Escudo", "Poção"]

# Acessar elementos pelo índice
print(inventario[0])  # Saída: Espada
print(inventario[2])  # Saída: Poção
print(len(inventario))  # Saída: 3

Adicionar itens com .append()

adicionar_itens.py 
mochila = ["Livro"]

# Adicionar novos itens ao final da lista
mochila.append("Lanterna")
mochila.append("Corda")

print(mochila)  # Saída: ['Livro', 'Lanterna', 'Corda']

Remover itens com .pop() e .remove()

remover_itens.py 
itens = ["Pão", "Queijo", "Maçã"]

# .pop() remove e retorna o último item
ultimo = itens.pop()
print(ultimo)  # Saída: Maçã

# .remove() remove um item específico pelo valor
itens.remove("Pão")
print(itens)  # Saída: ['Queijo']

Iterar sobre listas com for

iterar_lista.py 
nomes = ["Ana", "Bruno", "Carlos"]

# Forma 1: Iterar sobre os valores
for nome in nomes:
    print(f"Olá, {nome}!")

# Forma 2: Iterar com índice
for i in range(len(nomes)):
    print(f"{i}: {nomes[i]}")

2. Manipulando o Inventário

Dominar o inventário exige saber como colocar, retirar e procurar itens. Veja como o Python simplifica estas tarefas arcanas:

> Adicionar com Validação (if + input + operators)

adicionar_itens.py 
mochila = ["Espada"]
novo_item = input("O que encontraste no baú? ")

# Verificando se o item já existe ou se é proibido
if novo_item == "Veneno" or novo_item == "Amaldiçoado":
    print("Perigo! Não podes guardar itens malignos.")
elif novo_item in mochila:
    print("Já tens este item na mochila!")
else:
    mochila.append(novo_item)
    print(f"{novo_item} guardado com sucesso!")

> Remover e Usar Itens (pop + remove)

remover_itens.py 
inventario = ["Pão", "Poção de Vida", "Tocha"]

if len(inventario) > 0:
    # .pop() remove o último item e guarda-o numa variável
    item_usado = inventario.pop()
    print(f"Usaste a {item_usado} para iluminar o caminho!")
else:
    print("Mochila vazia! Não tens nada para usar.")

> Evolução de Equipamento (Index Access)

upgrade_item.py 
armas = ["Adaga de Madeira", "Arco Velho"]
ouro = int(input("Quanto ouro tens? "))

# Upgrade se tiver mais de 100 moedas
if ouro >= 100:
    print("A evoluir a tua Adaga...")
    armas[0] = "Espada de Aço" # Substitui o valor no índice 0
    print(f"Novo arsenal: {armas}")
else:
    print("Ouro insuficiente para o ferreiro.")

3. Laboratório: Integração Máxima

01 O Baú de Itens (Listas + While + If)

O Código Mestre!

Analise como o laço interage com a lista de forma dinâmica.

bau_tesouros.py 
inventario = []
CAPACIDADE_MAX = 5

while len(inventario) < CAPACIDADE_MAX:
    item = input("Encontraste um item! O que é? (ou 'sair'): ")
    
    if item.lower() == 'sair':
        break
    
    inventario.append(item)
    print(f"[{item}] guardado com sucesso!")

print("\nInventário Final:")
for i, it in enumerate(inventario):
    print(f"{i+1}. {it}")

02 Mercado de Ouro (Listas + Aritmética + For)

mercado.py 
precos = []
soma_total = 0.0

print("--- CALCULADORA DO MERCADOR ---")
qtd = int(input("Quantos itens vais vender? "))

for i in range(qtd):
    p = float(input(f"Preço do item {i+1}: "))
    precos.append(p)
    soma_total += p

print(f"\nValor Total: {soma_total:.2f} moedas.")
if soma_total > 1000 and len(precos) < 3:
    print("Vendeste itens de luxo! Poucos itens, muito ouro.")
