Dê poder de escolha aos seus scripts. Aprenda como o Python toma decisões usando apenas dois pontos e o poder dos espaços.

1. A Lei dos Dois Pontos

Em Portugol, usavas { } para dizer onde um bloco começava e terminava. Em Python, isso não existe. A linguagem usa dois pontos (:) para iniciar o bloco e a indentação (4 espaços) para saber o que está lá dentro.

O Jeito Portugol
se (condicao) {
  escreva("Olá")
}
O Jeito Python
if condicao:
    print("Olá")

2. If, Else e o Poderoso Elif

if (Se)

"Se isto for verdade, faz isto."

elif (Senão Se)

"Se a anterior falhou, mas isto for verdade, faz isto."

else (Senão)

"Se tudo o resto falhou, faz isto por defeito."

3. Laboratório: O Portal de Verificação

01 Radar de Velocidade (Entrada de Dados + If)

Simplicidade Python!

Utilizamos o input() para receber a velocidade.

radar.py 
# Recebendo dados do utilizador
velocidade = float(input("Velocidade da carroça (km/h): "))

if velocidade > 80:
    print("MULTADO! Estás a correr demais.")
    multa = (velocidade - 80) * 7
    print(f"Multa: {multa:.2f} moedas.")

print("Boa viagem!")

02 Classificador de Herói (Múltiplos Inputs)

rank_heroi.py 
nome = input("Nome do herói: ")
xp = int(input("Quanto XP acumulaste? "))

if xp < 1000:
    rank = "Aprendiz"
elif xp < 5000:
    rank = "Guerreiro"
else:
    rank = "Lenda"

print(f"O aventureiro {nome} é um: {rank}!")
