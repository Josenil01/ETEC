Módulo 1 Verificação de Lógica

Aula 05: O Oráculo da Verdade

Torne-se o próprio processador. Aprenda a rastrear o caminho dos dados e a caçar erros antes mesmo de o código ser executado.

1. O Rastro das Variáveis

Um Teste de Mesa é um processo manual onde o programador simula a execução de um algoritmo linha por linha, anotando o valor de cada variável num papel (ou tabela).

É a ferramenta definitiva para:

  • Entender como o computador processa os dados.
  • Encontrar erros de lógica que a IDE não deteta.
  • Verificar se as fórmulas matemáticas estão corretas.

3. Provas do Observador (Exercícios)

Nível: Aprendiz de Analista

01

Faça o teste de mesa do código abaixo no caderno:

x = 5; y = 10; x = x + y; y = x - 5; escreva(y)
02

Qual o valor final da variável resultado?

a = 2; b = 4; c = 8; resultado = (c / b) * a

Nível: Caçador de Bugs

03

Troca de Valores: Rastreie este algoritmo de troca:

A = 20; B = 50; TEMP = A; A = B; B = TEMP

Quais os valores finais de A e B?

04

Rastreie o comportamento das variáveis lógicas:

p = verdadeiro; q = falso; r = (p e q) ou (nao q)

Nível: Grão-Mestre da Memória

05

O Enigma Matemático: Realize o teste de mesa completo para:

val = 100; val = val / 2; val = val % 3; res = (val == 2) e (val != 0)

Qual o valor lógico final da variável res? Justifique com a tabela de rastreio.
