Aula 03: Variáveis e Memória

Aprenda a criar "espaços mágicos" na memória do computador para guardar as estatísticas e itens do seu herói.

1. O Que é uma Variável?

Pense numa variável como uma caixa com etiqueta dentro do computador. Quando você diz ao computador que quer guardar o nome do seu personagem, ele reserva um espacinho na memória RAM (a etiqueta) e coloca o valor lá dentro.

Para usar essas caixas, precisamos de três coisas:

  • Tipo: Define o que a caixa pode guardar (números, textos, etc).
  • Nome (Identificador): A etiqueta para encontrarmos a caixa depois.
  • Valor: O conteúdo que está dentro da caixa.

2. Os Tipos de Dados no Portugol

inteiro Números sem casas decimais (Ex: 10 vidas, nível 5).
real Números com vírgula (Ex: 1.75 de altura, 50.5 moedas).
cadeia Textos longos ou frases (Ex: "Arthur", "Espada de Fogo").
caracter Um único símbolo ou letra (Ex: 'M', 'F', '+').
logico Valores de Verdadeiro ou Falso (Ex: tem_chave = verdadeiro).

3. Exemplos Práticos por Tipo de Dado

Contador de Vidas (inteiro)

exemplo_inteiro.por 
programa {
  funcao inicio() {
    inteiro vidas = 3
    escreva("Você tem ", vidas, " vidas restantes.\n")
  }
}

Bolsa de Ouro (real)

exemplo_real.por 
programa {
  funcao inicio() {
    real moedas = 45.50
    escreva("Seu saldo atual é: ", moedas, " moedas.\n")
  }
}

Nome do Equipamento (cadeia)

exemplo_cadeia.por 
programa {
  funcao inicio() {
    cadeia arma = "Espada de Fogo"
    escreva("Você equipou: ", arma, "\n")
  }
}

Sigla de Classe (caracter)

exemplo_caracter.por
programa {
  funcao inicio() {
    caracter classe = 'M'
    escreva("Sua classe é: ", classe, "\n")
  }
}

Estado de Missão (logico)

exemplo_logico.por 
programa {
  funcao inicio() {
    logico em_missao = verdadeiro
    escreva("O herói está em missão? ", em_missao, "\n")
  }
}

Exemplo Integrado: Ficha de Criação de Herói

Sistema Interativo!

Agora vamos usar a função leia() para que o jogador preencha o seu próprio inventário.

criacao_personagem.por 
programa {
                                funcao inicio() {
                                    // 1. DECLARAÇÃO: Preparando as caixas da memória
                                    cadeia nome_heroi
                                    inteiro nivel, vida_base
                                    real ouro_inicial
                                    caracter sigla_classe // 'G' para Guerreiro, 'M' para Mago
                                    logico missao_ativa

                                    // 2. ENTRADA DE DADOS: Perguntando ao usuário
                                    escreva("--- BEM-VINDO À GUILDA DE AVENTUREIROS ---\n")
                                    
                                    escreva("Digite o nome do seu Herói: ")
                                    leia(nome_heroi)

                                    escreva("Qual o nível inicial do herói? ")
                                    leia(nivel)

                                    escreva("Quantas moedas de ouro ele carrega? ")
                                    leia(ouro_inicial)

                                    escreva("Qual a sigla da classe? (G/M/A): ")
                                    leia(sigla_classe)

                                    // 3. PROCESSAMENTO: Atribuição automática
                                    vida_base = 100
                                    missao_ativa = verdadeiro

                                    // 4. SAÍDA DE DADOS: Mostrando a Ficha Completa
                                    escreva("\n--- FICHA DO PERSONAGEM GERADA ---\n")
                                    escreva("NOME: ", nome_heroi, " (Classe: ", sigla_classe, ")\n")
                                    escreva("NÍVEL ATUAL: ", nivel, "\n")
                                    escreva("VIDA TOTAL: ", vida_base, " HP\n")
                                    escreva("TESOURO: ", ouro_inicial, " moedas\n")
                                    escreva("EM AVENTURA: ", missao_ativa)
                                }
                                }

4. Provas de Habilidade

Nível: Aprendiz

01

Crie um programa que declare uma variável para cada tipo básico do Portugol (inteiro, real, cadeia, caracter e logico) e atribua valores a elas.

02

Troca de Cristais: Declare duas variáveis inteiras A e B. Atribua valores a elas. Depois, tente fazer com que o valor de A passe para B e o de B passe para A. (Dica: Você vai precisar de uma terceira "caixa" vazia!).

Nível: Cavaleiro

03

Cartão de Visitante: Crie um programa que use a função leia() para perguntar o Nome, a Cidade e o Reino do utilizador e depois imprima tudo numa frase bem organizada.

04

Previsão de Nível: Leia um número inteiro que representa o nível atual de um jogador e exiba qual será o nível dele quando ele subir de nível (sucessor) e qual era o nível anterior (antecessor).

Nível: Mestre Arcano

05

O Círculo Mágico: Crie um programa que peça ao utilizador o raio de um círculo (variável real). Calcule e exiba a área desse círculo. Considere PI = 3.1415. (Fórmula: Área = PI * raio * raio).
