📚 Resumo Teórico: Arrays vs Listas

No Java, não podemos simplesmente "acrescentar" dados a qualquer variável. Precisamos de coleções organizadas. A escolha entre um Array e um ArrayList depende da previsibilidade do seu jogo:

Arrays (Estáticos)

Tamanho fixo definido no nascimento. Se crias uma equipa para 4 heróis, ela terá sempre 4 lugares, nem mais, nem menos. São ultra-rápidos e eficientes.

ArrayList (Dinâmicos)

Tamanho flexível. Funcionam como uma mochila mágica: podes adicionar itens (`add`) ou remover (`remove`) a qualquer momento durante a execução.

1. A Sintaxe da Ordem

Declaração de Array
int[] numeros = new int[5];
// Índices: 0, 1, 2, 3, 4
numeros[0] = 100;
Declaração de ArrayList
ArrayList<String> lista = new ArrayList<>();
lista.add("Poção");
int tam = lista.size();

2. Laboratório de Estruturas

01 A Equipa Fixa (Array Estático)

No Java, os arrays são objetos. Usamos .length para saber quantos espaços foram reservados.

Equipa.java 
public class Equipa {
    public static void main(String[] args) {
        // Declaração com valores imediatos
        String[] herois = {"Arthur", "Elena", "Rolf"};

        System.out.println("Listando a Guilda:");

        // Uso do .length para saber o tamanho
        for (int i = 0; i < herois.length; i++) {
            System.out.println("Posição " + i + ": " + herois[i]);
        }
    }
}

02 A Mochila de Loot (ArrayList Dinâmico)

O ArrayList faz parte do Java Collections Framework. Precisamos de importar java.util.ArrayList.

Inventario.java 
import java.util.ArrayList;

public class Inventario {
    public static void main(String[] args) {
        // Criando uma lista que cresce sozinha
        ArrayList<String> mochila = new ArrayList<>();

        // Adicionando itens
        mochila.add("Espada de Ferro");
        mochila.add("Escudo de Madeira");
        mochila.add("Poção de HP");

        // Removendo por índice (remove o Escudo)
        mochila.remove(1);

        System.out.println("Itens na mochila: " + mochila);
        System.out.println("Total: " + mochila.size());
    }
}

3. Erros Arcanos: Quando o Contentor Falha

IndexOutOfBoundsException

Tentar aceder a um índice que não existe (ex: índice 5 num array de tamanho 5). Lembra-te: os índices param sempre em tamanho - 1.

Length vs size()

No Array Estático usamos a propriedade .length (sem parênteses). No ArrayList usamos o método .size(). Trocar estes dois é erro garantido.
