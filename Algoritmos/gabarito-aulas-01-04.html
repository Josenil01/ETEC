Gabarito - Algoritmos (Aulas 01 a 04)

5 questões teóricas + 5 práticas com respostas em formato claro para apoio em sala.

Conteúdo: algoritmo, fluxograma, variáveis, operadores e expressões

Parte A - Teóricas (Respostas esperadas)

1) O que é algoritmo?

Resposta

Algoritmo é uma sequência finita e organizada de passos para resolver um problema. Ele precisa ter início, processamento e fim, com instruções claras para evitar ambiguidades.

2) Sequência lógica correta x incorreta

Resposta

Sequência correta segue a ordem necessária das ações. Sequência incorreta troca a ordem e compromete o resultado.

  • Correta: ligar o fogão → colocar panela → aquecer água.
  • Incorreta: colocar macarrão no prato → depois cozinhar.

3) Símbolos de fluxograma

Resposta

  • Início/Fim: marca onde o processo começa e termina.
  • Processamento: representa cálculos e ações.
  • Entrada/Saída: leitura de dados e exibição de resultados.
  • Decisão: testa condição e cria caminhos (Sim/Não).

4) Variável, tipo e valor

Resposta

Variável é um espaço na memória com nome para armazenar dados. Tipo define que tipo de dado pode ser guardado (inteiro, real, cadeia, lógico). Valor é o conteúdo atual da variável.

5) Aritméticos, relacionais e lógicos

Resposta

  • Aritméticos: +, -, *, /, %.
  • Relacionais: ==, !=, >, <, >=, <=.
  • Lógicos: e, ou, nao.

7) Média de 3 notas com aprovado/reprovado

Resposta em Portugol

programa {
  funcao inicio() {
    real n1, n2, n3, media

    escreva("Nota 1: ")
    leia(n1)
    escreva("Nota 2: ")
    leia(n2)
    escreva("Nota 3: ")
    leia(n3)

    media = (n1 + n2 + n3) / 3
    escreva("Média = ", media, "\n")

    se (media >= 6) {
      escreva("Aprovado")
    } senao {
      escreva("Reprovado")
    }
  }
}

9) Classificação por faixa etária

Resposta em Portugol

programa {
  funcao inicio() {
    inteiro idade

    escreva("Digite a idade: ")
    leia(idade)

    se (idade >= 0 e idade <= 12) {
      escreva("Criança")
    } senao se (idade >= 13 e idade <= 17) {
      escreva("Adolescente")
    } senao se (idade >= 18 e idade <= 59) {
      escreva("Adulto")
    } senao {
      escreva("Idoso")
    }
  }
}

10) Controle de acesso lógico

Resposta em Portugol

programa {
  funcao inicio() {
    logico temChave, ehAdministrador
    inteiro nivel

    escreva("Tem chave? (verdadeiro/falso): ")
    leia(temChave)
    escreva("Nível: ")
    leia(nivel)
    escreva("É administrador? (verdadeiro/falso): ")
    leia(ehAdministrador)

    se ((temChave e nivel >= 5) ou ehAdministrador) {
      escreva("Acesso liberado")
    } senao {
      escreva("Acesso negado")
    }
  }
}
