Aula 17: Grimórios Externos

Não reinvente a roda. Aprenda a importar módulos e bibliotecas para usar funções poderosas criadas por outros programadores.

📚 Resumo Teórico

Um Módulo é simplesmente um ficheiro contendo código Python (funções, variáveis, classes). Quando o seu projeto cresce, não faz sentido ter tudo num único ficheiro.

O Python já vem com uma "Biblioteca Padrão" cheia de módulos úteis que você pode importar para o seu script. É como se o herói pudesse pedir emprestado um pergaminho mágico de outro reino.

  • import: Traz o módulo inteiro para o seu código.
  • from ... import: Traz apenas uma função específica do módulo.
  • as: Cria um apelido (alias) para o módulo, facilitando a escrita.

1. As Três Formas de Invocação

Dependendo da sua necessidade, existem diferentes formas de chamar o poder de um módulo:

Importação Geral import math

Acede como: math.sqrt()

Importação Específica from random import randint

Acede direto: randint()

Uso de Apelido (Alias) import datetime as dt

Acede como: dt.now()

2. Exemplos Práticos Comentados

01. O Oráculo Matemático (Módulo math)

alquimia_math.py 
import math

numero = float(input("Digite um valor para a poção: "))

# math.sqrt() -> Raiz Quadrada
raiz = math.sqrt(numero)

# math.ceil() -> Arredondar para cima
frascos = math.ceil(raiz)

print(f"O poder bruto é {raiz:.2f}, precisas de {frascos} frascos.")

02. Os Dados do Destino (Módulo random)

dados_rpg.py 
from random import randint, choice

# Gerar um número inteiro aleatório entre 1 e 20 (D20)
dado = randint(1, 20)
print(f"Rolaste um dado: {dado}")

# Escolher um item aleatório de uma lista (Aula 14)
loot = ["Espada", "Escudo", "Poção", "Anel Mágico"]

if dado == 20:
    print(f"CRÍTICO! Ganhaste um {item_ganho}!")
else:
    print("Nada de especial aconteceu.")

3. Laboratório: Simulador de Combate Dinâmico

Este programa integra Módulos, Funções e Lógica para criar um turno de batalha imprevisível.

arena_final.py 
import random
import time # Módulo para pausas

def atacar(nome_atacante, forca_base):
    print(f"{nome_atacante} está a preparar o ataque...")
    time.sleep(1) # Pausa de 1 segundo
    
    variacao = random.randint(-5, 10)
    dano_final = forca_base + variacao
    return max(0, dano_final)

# Início da Batalha
heroi_forca = 20
inimigo_vida = 50

while inimigo_vida > 0:
    dano = atacar("Arthur", heroi_forca)
    inimigo_vida -= dano
    print(f"Causaste {dano} de dano! Inimigo tem {max(0, inimigo_vida)} HP.")
    
print("VITÓRIA! O inimigo foi derrotado.")

4. Erros Arcanos: Quando a Invocação Falha

ModuleNotFoundError

Acontece quando tentas importar um módulo que não existe ou cujo nome escreveste errado.

import matematic
# Erro! O correto é 'math'

Conflito de Nomes

Nunca dês ao teu ficheiro .py o nome de uma biblioteca famosa (ex: não cries um random.py).

# Se o teu ficheiro se chama math.py,
# 'import math' vai importar a ti mesmo!
