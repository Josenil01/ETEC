Aula 15: Mapas e Chaves

Vá além das listas. Aprenda a usar Dicionários e Conjuntos para mapear informações complexas e garantir a unicidade dos seus dados.

📚 Resumo Teórico

Até agora, as nossas listas eram acedidas por números (índices). Mas e se quiséssemos encontrar a "Força" do herói sem saber a sua posição? Para isso usamos os Dicionários.

  • Dicionários (dict): Coleções de pares Chave: Valor. Perfeito para cadastros e perfis.
  • Conjuntos (set): Coleções que não permitem duplicatas e não têm ordem fixa. Ótimo para verificar itens únicos.

Nesta aula, integraremos: input(), if/elif/else, operadores lógicos, laços for e as novas estruturas.

1. O Livro de Registos (Dicionários)

Um dicionário usa chavetas {} e mapeia uma etiqueta (Chave) a um conteúdo (Valor).

Chave "Nome"
Valor "Aragorn"
Chave "Vida"
Valor 100

2. Exemplos Práticos Comentados

Manipulando Atributos (Dict + Update)

ficha_heroi.py 
# Criar dicionário de atributos
heroi = {
    "nome": "Arthur",
    "forca": 15,
    "classe": "Guerreiro"
}

# Aceder a um valor pela chave
print(f"O herói {heroi['nome']} tem {heroi['forca']} de força.")

# Modificar ou Adicionar novo par
heroi["forca"] += 5
heroi["ouro"] = 100

print(heroi)

Garantindo Itens Únicos (Sets)

colecao_rara.py 
# Conjuntos usam {} mas não têm pares (:)
minerais = {"Ferro", "Ouro", "Prata"}

# Tentar adicionar duplicado (não acontece nada)
minerais.add("Ferro")

# Verificar se item existe (Operador IN)
if "Ouro" in minerais:
    print("Tens ouro na tua coleção única!")

print(f"Coleção: {minerais}")

3. Laboratório: O Criador de Personagens

Neste laboratório, usamos input para preencher um dicionário, while para controlar o fluxo e if para validar bónus.

rpg_maker.py 
personagem = {}

# Entrada de Dados
personagem["nome"] = input("Nome do Herói: ")
personagem["nivel"] = int(input("Nível (1-10): "))
personagem["xp"] = 0

# Lógica de Bónus com IF e Operadores Aritméticos
if personagem["nivel"] > 5:
    personagem["status"] = "Veterano"
    personagem["vida"] = personagem["nivel"] * 20
else:
    personagem["status"] = "Novato"
    personagem["vida"] = personagem["nivel"] * 10

# Exibição Integrada
print("\n--- FICHA TÉCNICA ---")
for chave, valor in personagem.items():
    print(f"{chave.upper()}: {valor}")
