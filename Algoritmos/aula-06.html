Aula 06: As Encruzilhadas do Destino

Dê "cérebro" ao seu herói. Aprenda a usar as estruturas de decisão para que o seu código possa escolher caminhos baseados em condições.

1. O Poder da Escolha

Até agora, os nossos códigos eram lineares: executavam uma linha após a outra sem questionar. As Estruturas de Decisão (ou Condicionais) permitem que o programa avalie uma pergunta (condição) e execute blocos de código diferentes dependendo da resposta ser Verdadeira ou Falsa.

2. A Estrutura Se...Senao

Estrutura Simples (Se) "Se eu tiver moedas, compro a espada."

Executa um bloco apenas se a condição for verdadeira. Se for falsa, ignora e segue em frente.

Estrutura Composta (Se...Senao) "Se tiver mana, lanço fogo. Senão, uso o cajado."

Oferece um caminho alternativo obrigatório caso a primeira condição falhe.

3. Laboratório Prático de Decisões

01 Exemplo: O Portão da Mana (Condicional Composta)

Teste de Condição!

Verificamos se o mago tem mana suficiente para conjurar um feitiço.

feitico_mana.por 
programa {
  funcao inicio() {
    inteiro mana_atual, CUSTO = 40
    escreva("Quanto de mana tens? ")
    leia(mana_atual)

    se (mana_atual >= CUSTO) {
        escreva("Feitiço lançado! ✨")
        mana_atual = mana_atual - CUSTO
    } senao {
        escreva("Mana insuficiente... 🌑")
    }
    escreva("\nMana final: ", mana_atual)
  }
}

02 Exemplo: Radar das Estradas (Condicional Simples)

Multa Automática!

Se a velocidade for maior que o limite, aplicamos a multa.

radar_velocidade.por 
programa {
  funcao inicio() {
    real velocidade, limite = 80.0
    escreva("Velocidade do veículo (km/h): ")
    leia(velocidade)

    se (velocidade > limite) {
        escreva("ALERTA: Velocidade excedida! Multa aplicada. 🚔")
    }
    escreva("\nBoa viagem! Conduza com cuidado.")
  }
}

03 Exemplo: Classificador de Rank (Encadeamento / Senao Se)

Evolução de Rank!

Usamos múltiplos níveis de decisão para classificar o aventureiro.

rank_heroi.por 
programa {
  funcao inicio() {
    inteiro xp
    escreva("Digite a sua Experiência (XP): ")
    leia(xp)

    se (xp < 1000) {
        escreva("Rank: Bronze 🛡️")
    } senao se (xp < 3000) {
        escreva("Rank: Prata ⚔️")
    } senao se (xp < 6000) {
        escreva("Rank: Ouro 🏆")
    } senao {
        escreva("Rank: Lendário 💎")
    }
  }
}

Nível: Aprendiz

01

Par ou Ímpar: Peça um número ao utilizador e diga se ele é par ou ímpar (Dica: use o resto da divisão por 2).

02

Aprovação do Aluno: Leia uma nota de 0 a 10. Se for maior ou igual a 7, escreva "Aprovado", senão escreva "Reprovado".

Nível: Cavaleiro

03

Duelo de Titãs: Leia dois números diferentes e exiba qual deles é o maior.

04

Ano Bissexto: Crie um programa que leia um ano e verifique se ele é bissexto (divisível por 4).

Nível: Mestre Arcano

05

Calculadora de Runas: Peça dois números e um caracter (+, -, *, /). Use estruturas de decisão para realizar a operação matemática correta baseada no símbolo digitado.
