Módulo 1 Lógica de Controlo II

Aula 07: O Menu de Escolhas

Organize as opções do seu destino. Aprenda a usar o Escolha-Caso para criar menus profissionais e eficientes.

1. Porquê usar o Escolha-Caso?

Imagine que precisa de criar um menu com 10 opções. Usar se...senao se repetidamente tornaria o código confuso e difícil de ler. O Escolha-Caso (também conhecido como Switch) é uma estrutura desenhada especificamente para testar uma única variável contra vários valores fixos.

Regra de Ouro: O Escolha-Caso só funciona com valores exatos (inteiros ou caracteres). Não serve para intervalos como "se x for maior que 10".

2. Anatomia do Escolha-Caso

escolha

Inicia a estrutura avaliando uma variável.

caso

Compara a variável com um valor específico.

pare

Obrigatório! Interrompe a execução para não entrar no próximo caso.

outro_caso

Executado se nenhum dos valores anteriores coincidir.

3. Laboratório Prático de Menus

01 Exemplo: O Menu da Taverna (Inteiros)

Aventuras Gastronómicas!

Usamos números para escolher o prato do dia.

menu_taverna.por 
programa {
  funcao inicio() {
    inteiro opcao
    escreva("1. Hidromel\n2. Carne de Javali\n3. Sopa de Ervas\n")
    escreva("Escolha o seu pedido: ")
    leia(opcao)

    escolha (opcao) {
      caso 1:
        escreva("A caminho! Cuidado com a ressaca.")
        pare
      caso 2:
        escreva("Um banquete digno de um rei!")
        pare
      caso 3:
        escreva("Leve e saudável para a viagem.")
        pare
      outro_caso:
        escreva("Opção inválida! O taberneiro olha de lado.")
    }
  }
}

02 Exemplo: Calculadora Arcaica (Caracteres)

Magia Matemática!

Desta vez, usamos símbolos matemáticos como caracteres para decidir a operação.

calculadora_runica.por 
programa {
  funcao inicio() {
    real n1, n2
    caracter operacao

    escreva("Digite dois números: ")
    leia(n1, n2)
    escreva("Escolha a operação (+, -, *, /): ")
    leia(operacao)

    escolha (operacao) {
      caso '+':
        escreva("Soma: ", n1 + n2)
        pare
      caso '-':
        escreva("Subtração: ", n1 - n2)
        pare
      caso '*':
        escreva("Multiplicação: ", n1 * n2)
        pare
      caso '/':
        escreva("Divisão: ", n1 / n2)
        pare
      outro_caso:
        escreva("Símbolo arcano desconhecido!")
    }
  }
}

4. Provas de Organização

Nível: Aprendiz

01

Saudação Poliglota: Crie um programa que peça um número de 1 a 3 e exiba "Bom dia" em 3 línguas diferentes.

02

Calendário Rápido: Leia um número de 1 a 12 e exiba o nome do mês correspondente.

Nível: Cavaleiro

03

Balança Interplanetária: Leia o peso de um objeto e um código de planeta (1-Mercúrio, 2-Vénus, 3-Marte). Calcule o peso nesse planeta usando as gravidades correspondentes.

04

Classificação de Torneio: Leia a idade de um competidor e informe a sua categoria: 5-7 (Infantil A), 8-10 (Infantil B), 11-13 (Juvenil A). Use Escolha-Caso.

Nível: Mestre Arcano

05

Simulador de Caixa Eletrónico: Crie um menu com as opções: 1-Ver Saldo, 2-Depositar, 3-Sacar e 4-Sair. Utilize uma variável para o saldo e atualize-a conforme a escolha do utilizador. Lembre-se de não permitir saques maiores que o saldo!
