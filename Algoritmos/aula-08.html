Módulo 1 Laços de Repetição

Aula 08: O Treino Infinito

Não te canses com tarefas repetitivas. Aprenda a usar o laço "Enquanto" para dominar a arte da iteração.

1. O Conceito de Iteração

Até agora, se quiséssemos que o computador escrevesse "Olá" 5 vezes, teríamos de escrever 5 linhas de código. Mas e se quiséssemos 1000 vezes? Ou até que o utilizador diga "Parar"?

Para isso usamos os Laços de Repetição (ou Loops). O laço Enquanto (While) repete um bloco de código enquanto uma condição for verdadeira.

Atenção: Deves garantir que a condição se torne falsa em algum momento, caso contrário criarás um "Loop Infinito" que pode travar o programa!

2. Contadores e Acumuladores

Contador

"Quantas flexões já fiz?"

Uma variável que aumenta de 1 em 1 (ex: c = c + 1) para contar voltas.

Acumulador

"Quanto ouro total tenho no baú?"

Uma variável que soma valores variáveis (ex: soma = soma + novo_valor).

3. Laboratório de Treino Dinâmico

01 O Campo de Treino (Laço Básico)

treino_guerreiro.por 
inteiro contador = 1, meta
escreva("Quantas flexões hoje? ")
leia(meta)

enquanto (contador <= meta) {
    escreva(contador, "ª flexão...\n")
    contador = contador + 1
}
escreva("Treino concluído!")

02 Saque na Masmorra (Repetição + IF)

Neste cenário, o herói continua a pegar itens enquanto houver espaço na mochila, usando um se para validar cada item.

saque_masmorra.por 
programa {
  funcao inicio() {
    inteiro espaco_mochila = 20, peso_item
    
    enquanto (espaco_mochila > 0) {
        escreva("Espaço livre: ", espaco_mochila, "kg. Peso do novo item: ")
        leia(peso_item)
        
        se (peso_item <= espaco_mochila) {
            espaco_mochila = espaco_mochila - peso_item
            escreva("Item guardado com sucesso!\n")
        } senao {
            escreva("Muito pesado! Tenta algo mais leve.\n")
        }
    }
    escreva("Mochila cheia! Hora de voltar à cidade.")
  }
}

03 Simulador de Batalha (Repetição + Switch Case)

O laço permite repetir o menu de combate até que o herói decida "Fugir".

combate_menu.por 
programa {
  funcao inicio() {
    inteiro opcao = 0
    
    enquanto (opcao != 3) {
        escreva("\n--- TURNO DE COMBATE ---\n")
        escreva("1. Atacar\n2. Defender\n3. Fugir\nEscolha: ")
        leia(opcao)
        
        escolha (opcao) {
            caso 1:
                escreva("Causaste 10 de dano!\n")
                pare
            caso 2:
                escreva("Bloqueaste o ataque inimigo!\n")
                pare
            caso 3:
                escreva("Correste covardemente...\n")
                pare
            outro_caso:
                escreva("Comando inválido!\n")
        }
    }
  }
}

4. Provas de Resistência

Nível: Recruta

01

Pares do Reino: Crie um programa que exiba todos os números pares de 0 até 50 usando o laço enquanto.

02

Sentinela de Senha: Peça uma palavra-passe ao utilizador. Enquanto a senha for incorreta, continue a pedir a senha e exiba "Acesso Negado".

Nível: Guerreiro

03

Somatório de Tesouros: Crie um programa que leia 10 números (moedas encontradas) e, ao final, exiba a soma total acumulada.

04

Tabuada Mágica: Peça um número ao utilizador e exiba a tabuada completa desse número (de 1 a 10) usando um laço de repetição.

Nível: Mestre de Armas

05

Adivinhação de Monstros: O programa deve definir um número secreto (ex: 7). O utilizador deve tentar adivinhar. Enquanto não acertar, o programa diz se o palpite foi muito alto ou muito baixo e conta quantas tentativas foram feitas até o acerto final.
