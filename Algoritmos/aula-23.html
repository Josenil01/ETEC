Módulo 2 Repetições em Java

Aula 23: Ciclos de Ferro

Repetir é a base da automação. Aprende a usar o while, o for e o robusto do-while para forjar processos contínuos.

📚 Resumo Teórico

Em Java, os laços de repetição (loops) permitem executar o mesmo bloco de código múltiplas vezes. Tal como no Portugol, usamos chavetas para definir o corpo do laço.

  • While: Executa enquanto a condição for verdadeira. Testa antes de entrar.
  • For: Ideal para contagens. Concentra inicialização, condição e incremento numa só linha.
  • Do-While: O teimoso. Executa o código e só depois testa a condição. Garante pelo menos uma execução.

1. A Trindade da Repetição

O Clássico For
for (int i = 0; i < 10; i++) {
    // Repete 10 vezes
}
O Flexível While
while (condicao) {
    // Repete se OK
}
O Robusto Do-While
do {
    // Faz primeiro
} while (condicao);

2. Laboratório de Automação

01. Campo de Treino (Laço For)

Usamos o for quando sabemos o limite exato. No Java, o incremento i++ é a forma curta de i = i + 1.

Treino.java 
public class Treino {
    public static void main(String[] args) {
        
        System.out.println("Iniciando 10 flexões de braço...");

        for (int i = 1; i <= 10; i++) {
            System.out.println(i + "ª flexão concluída!");
        }

        System.out.println("Treino finalizado, recruta!");
    }
}

02. Menu da Alquimia (Do-While + Switch)

O do-while é excelente para menus, pois garante que o utilizador veja as opções pelo menos uma vez antes de decidir sair.

LojaAlquimia.java 
import java.util.Scanner;

public class LojaAlquimia {
    public static void main(String[] args) {
        Scanner leitor = new Scanner(System.in);
        int opcao;

        do {
            System.out.println("\n[1] Poção de Vida | [2] Poção de Mana | [0] Sair");
            System.out.print("Escolha a sua compra: ");
            opcao = leitor.nextInt();

            switch (opcao) {
                case 1: System.out.println("Saúde recuperada!"); break;
                case 2: System.out.println("Energia arcana carregada!"); break;
                case 0: System.out.println("A fechar a loja..."); break;
                default: System.out.println("Opção inexistente no grimório.");
            }
        } while (opcao != 0);

        leitor.close();
    }
}

3. Erros Arcanos: O Ciclo Sem Fim

Loop Infinito

Acontece quando esqueces de atualizar a variável de controlo (ex: esquecer o i++). O programa ficará "preso" para sempre.

Ponto e Vírgula no Do-While

Um erro clássico do Java: o do-while é o único laço que EXIGE um ponto e vírgula após o parêntese do while(...);.

4. Desafios da Forja

Nível Recruta

Contagem de Moedas

Cria um programa que use um for para mostrar os números de 10 a 1 (contagem decrescente) e termina com "Fogo!".

Nível Explorador

Somatório de Experiência

Peça 5 valores de XP ao utilizador usando um while ou for. Ao final, exiba a soma total de XP acumulado pela equipa.

Nível Mestre

Validador de Entrada

Usa um do-while para pedir uma nota entre 0 e 10. Se o utilizador digitar algo fora desse intervalo, o programa deve repetir o pedido até que a nota seja válida.
