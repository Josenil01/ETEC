Aula 05: Manipulação de Strings

Strings são mais do que simples texto. Aprenda a fatiar, transformar e extrair informações cruciais de sequências de caracteres.

1. Arquitetura de Texto

Como o Python entende e manipula cadeias de caracteres.

Índices

Cada letra numa String tem uma morada (índice). O primeiro caractere é sempre o 0. Índices negativos (ex: -1) começam do fim.

Fatiamento (Slicing)

Podes extrair partes de uma string usando a sintaxe [início:fim:passo]. É essencial para limpar dados de ficheiros ou URLs.

Imutabilidade

Strings são imutáveis. Não podes alterar uma letra diretamente, mas podes criar uma nova string transformada a partir da original.

2. Exemplo Prático: Limpeza de Logs

Simulação de tratamento de dados vindo de um sistema de sensores industriais.

LOG_PROCESSOR.PY 
# --- SISTEMA DE PROCESSAMENTO DE DADOS TÉCNICOS ---

# Exemplo de uma string suja vinda de um sensor
log_bruto = "   ID_2026-SECTOR_A4-TEMP_75C   "

# 1. TRATAMENTO INICIAL (Remover espaços vazios nas pontas)
log_limpo = log_bruto.strip()

# 2. FATIAMENTO (Extrair partes específicas)
id_sistema = log_limpo[3:7]      # Extrai '2026'
setor = log_limpo[8:16]           # Extrai 'SECTOR_A'

# 3. MÉTODOS DE TRANSFORMAÇÃO
print("Log em Maiúsculas:", log_limpo.upper())
print("Log em Minúsculas:", log_limpo.lower())

# Substituir caracteres
log_formatado = log_limpo.replace("_", " ")
print("Log Formatado:", log_formatado)

# 4. DIVISÃO DE DADOS (Split)
# Transforma uma string numa lista de pedaços baseada num separador
componentes = log_limpo.split("-")
print("\nLista de Componentes:", componentes)

# 5. VERIFICAÇÃO DE CONTEÚDO
tem_setor_a = "SECTOR_A" in log_limpo
print(f"\nO log pertence ao Setor A? {tem_setor_a}")

# 6. INVERSÃO DE STRING (O truque do fatiamento)
codigo_secreto = "TX-990"
codigo_invertido = codigo_secreto[::-1]
print(f"Código Reverso: {codigo_invertido}")
