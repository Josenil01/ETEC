Aula 08: O Ciclo While

Aprenda a criar blocos de código que se repetem enquanto uma condição for verdadeira. Essencial para menus, validações e jogos.

1. Loops Infinitos e Controlados

O conceito de "Enquanto" aplicado ao fluxo do software.

Estrutura While

O while (enquanto) verifica a condição no topo. Se for True, executa o bloco e volta ao topo para verificar novamente.

A Importância da Paragem

Para evitar um Loop Infinito (que trava o PC), precisamos de algo dentro do loop que mude a condição para False eventualmente.

Contadores

Usamos variáveis como i = i + 1 para contar quantas vezes o loop rodou, servindo como um interruptor de segurança.

2. Exemplo Prático: Monitor de Energia

Sistema profissional que monitoriza o carregamento de uma bateria até atingir o limite seguro.

BATTERY_MONITOR.PY 
# --- SISTEMA DE CARREGAMENTO AUTOMATIZADO ---

carga_atual = 0
limite_seguranca = 100
taxa_carregamento = 15

print("--- INICIANDO CARREGAMENTO DA BATERIA ---")

# O loop continuará enquanto a carga for menor que o limite
while carga_atual < limite_seguranca:
    print(f"Status: {carga_atual}% ... Carregando...")
    
    # Incremento: Essencial para não criar loop infinito
    carga_atual = carga_atual + taxa_carregamento
    
    # Pequena correção para não exibir mais de 100%
    if carga_atual > 100:
        carga_atual = 100

print(f"Status Final: {carga_atual}%")
print("--- CARGA COMPLETA. SISTEMA SEGURO. ---")

# Exemplo de loop com entrada do utilizador
opcao = ""
while opcao.lower() != "sair":
    opcao = input("\nDigite algo (ou 'sair' para finalizar): ")
    print(f"Tu digitaste: {opcao}")
