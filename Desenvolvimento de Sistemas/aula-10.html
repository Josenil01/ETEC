Aula 10: Interrupções e Diagnóstico

Domine os comandos de escape break e continue, e aprenda a usar o Debugger para ler a "mente" do seu computador.

1. Arquitetura de Controlo

Mecanismos para desviar o fluxo padrão de repetição e encontrar erros.

Comando break

O "Eject" do laço. Encerra a repetição instantaneamente, ignorando qualquer linha restante abaixo dele dentro do loop.

Comando continue

O "Skip" da iteração. Interrompe apenas o passo atual e força o Python a voltar ao topo do loop para o próximo item.

Watch & Variable

No modo Debug, podes observar variáveis específicas mudarem de valor a cada segundo, facilitando a caça aos bugs.

Guia Rápido: Debugging no VS Code

1. O Breakpoint: Clica à esquerda do número de qualquer linha. Um ponto vermelho aparecerá. O código vai parar exatamente ali.
2. Iniciar (F5): Escolhe "Python Debugger". A barra de ferramentas flutuante aparecerá no topo.
3. Passo a Passo (F10): Avança uma linha de cada vez e observa as variáveis no painel da esquerda (Variables).
4. Parar (Shift+F5): Encerra a sessão de diagnóstico.

2. Exemplo: Firewall Inteligente

Sistema que analisa pacotes, ignora ruído e bloqueia IPs suspeitos instantaneamente.

Advanced_Security.py 
# --- SISTEMA DE VARREDURA DE SEGURANÇA PRO ---

print("INICIANDO FIREWALL NÍVEL 3...")
pacotes_analisados = 0

for id_pacote in range(100, 121):
    print(f"\nChecando pacote {id_pacote}...")
    
    # Regra 1: Ignorar pacotes de rotina (IDs pares)
    if id_pacote % 2 == 0:
        print("STATUS: Tráfego de rotina. Saltando...")
        continue # Volta para o topo do loop

    # Regra 2: Detetar Pacote Malicioso Crítico
    if id_pacote == 115:
        print("ALERTA: Malware detetado no ID 115!")
        print("BLOQUEANDO TODO O TRÁFEGO IMEDIATAMENTE.")
        break # Encerra o loop for completamente

    print(f"ID {id_pacote}: Verificação profunda concluída.")
    pacotes_analisados += 1

print("\n" + "=" * 30)
print(f"Relatório Final: {pacotes_analisados} pacotes seguros registados.")
print("Sessão de segurança encerrada.")

Aviso: O código acima é para estudo visual. Digite no VS Code para ativar o seu raciocínio lógico.
