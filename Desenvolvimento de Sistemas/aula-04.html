Aula 04: Entrada de Dados e Casting

Torne seus programas interativos. Aprenda a capturar informações do teclado e converter tipos de dados corretamente.

1. Conceitos de Interatividade

input() A Função de Entrada

O comando que "pausa" o programa e espera o utilizador digitar algo e apertar ENTER.
Atenção: O input sempre devolve o que foi digitado como Texto (String), mesmo que sejam números.

int() float() Casting (Conversão)

Como o input retorna texto, não podemos fazer matemática direta ("10" + "10" vira "1010"). Precisamos converter manualmente:

  • int(): Converte para Inteiro.
  • float(): Converte para Decimal.

2. Exemplo Prático: Formulário RH

user_data.py

Este script simula um cadastro simples, demonstrando o erro comum de não converter números e a solução correta.

# --- CADASTRO DE FUNCIONÁRIO (Entrada de Dados) ---

print("--- SISTEMA DE RH v1.0 ---")

# 1. ENTRADA DE TEXTO (Padrão)
# O cursor ficará esperando na mesma linha devido ao espaço no final da string
nome = input("Digite o nome completo: ")

# 2. ENTRADA DE NÚMEROS (CASTING)
# Erro comum: idade = input("Idade: ") -> Isso gera erro se tentarmos somar depois
# Forma correta: Envolver o input() com int() ou float()

idade = int(input("Digite a idade: "))
salario_atual = float(input("Digite o salário atual: "))

# 3. PROCESSAMENTO DE DADOS
# Agora podemos fazer matemática porque convertemos os tipos acima
ano_aposentadoria = 2026 + (65 - idade)
novo_salario = salario_atual * 1.10  # Aumento de 10%

# 4. SAÍDA FORMATADA (F-Strings)
# O 'f' antes das aspas permite colocar variáveis dentro de {chaves}
print("\n--- FICHA GERADA ---")
print(f"Funcionário: {nome}")
print(f"Idade registada: {idade} anos")
print(f"Ano estimado de reforma: {ano_aposentadoria}")
print(f"Salário projetado: R$ {novo_salario:.2f}") 
# O :.2f formata para 2 casas decimais

3. Missões no Aranhaverso

Tema: Spider-Man - Utilize input() para coletar dados das variantes aranha e int()/float() para calibrar os lançadores de teia.

Nível: Fácil (Amigável Vizinhança)

MISSÃO 01: IDENTIDADE

Crie um programa que pergunte: "Qual o seu nome de Aranha?" e "De qual Terra você é?" (Ex: Terra-616). No final, imprima uma mensagem de boas-vindas à Sociedade Aranha.

MISSÃO 02: ANO DA MORDIDA

Pergunte ao utilizador: "Em que ano você foi picado?" (Converta para int). Calcule há quantos anos ele é o Homem-Aranha (Use o ano atual 2026) e exiba o resultado.

MISSÃO 03: MULTIVERSO

Pergunte a idade de Miles Morales e a idade de Peter B. Parker. Some as duas idades e exiba: "Juntos vocês têm X anos de experiência".

MISSÃO 04: LANÇADOR DE TEIA

2. "Gasto por balançada (ml):" (float)
Calcule quantas vezes o Aranha pode se balançar antes de acabar a teia (Capacidade / Gasto).
