Combine condições com and, or e aprenda a nova estrutura match case para simplificar sistemas complexos.

1. Operadores Lógicos e Seleção

Ferramentas para validar estados simultâneos e gerir opções.

and / or / not

and: Só é verdade se TUDO for verdade.
or: É verdade se PELO MENOS UM for verdade.
not: Inverte o resultado (Verdade vira Falso).

match case

Introduzido no Python 3.10, é uma forma elegante de comparar uma variável contra vários valores específicos. Substitui longas sequências de if/elif.

2. Exemplo Prático: Controlo de Acessos

Sistema que valida o nível de segurança e o cargo do utilizador.

ACCESS_SECURITY.PY 
# --- SISTEMA DE GESTÃO DE ACESSOS EMPRESARIAIS ---

cargo = input("Cargo (Admin/User/Guest): ").capitalize()
tentativas = int(input("Tentativas de login hoje: "))
sistema_ativo = True

# 1. USO DE OPERADORES LÓGICOS
# Regra: Só entra se for Admin E tiver menos de 5 tentativas
if cargo == "Admin" and tentativas < 5:
    print("\n[ACESSO TOTAL CONCEDIDO]")
elif cargo == "User" or cargo == "Admin":
    print("\n[ACESSO RESTRITO CONCEDIDO]")
else:
    print("\n[ACESSO NEGADO]")

# 2. USO DE MATCH CASE (Para seleções diretas)
print("\n--- CONFIGURAÇÕES DE PERFIL ---")
match cargo:
    case "Admin":
        print("Painel: Editar, Apagar, Criar.")
    case "User":
        print("Painel: Visualizar, Editar Próprio.")
    case "Guest":
        print("Painel: Apenas Leitura.")
    case _:  # Caso padrão (default)
        print("Cargo não identificado no sistema.")
