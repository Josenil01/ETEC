Módulo 2 Iteração Definida

Aula 09: O Laço For e a Função Range

Domine a estrutura de repetição baseada em sequências e intervalos. Aprenda a processar listas e cadeias de caracteres de forma eficiente.

1. Conceitos de Iteração

Estrutura For

Diferente do while, o for é usado quando sabemos exatamente quantas vezes queremos repetir ou quando queremos percorrer todos os itens de uma coleção (como as letras de uma string).

A Função range()

Gera uma sequência numérica.
range(10): gera de 0 a 9.
range(5, 15): gera de 5 a 14.
range(0, 20, 2): gera de 0 a 18 pulando de 2 em 2.

2. Exemplo Prático: Automação de Inventário

inventory_check.py

Este script demonstra como gerar IDs sequenciais e processar caracteres individuais de um código de segurança.

# --- SISTEMA DE GERAÇÃO DE ETIQUETAS E SEGURANÇA ---

print("--- INICIANDO PROCESSAMENTO DE LOTES ---")

# 1. USANDO RANGE PARA GERAÇÃO SEQUENCIAL
# Simulação: Gerar IDs para o lote de 101 a 110
for numero in range(101, 111):
    print(f"Gerando etiqueta para o produto ID: {numero}")

print("\n--- VALIDAÇÃO DE CÓDIGO DE ACESSO ---")

# 2. ITERANDO SOBRE UMA STRING
codigo_seguranca = "AX-779"

print("Analisando caracteres do código:")
for caractere in codigo_seguranca:
    print(f"Validando componente: {caractere}")

# 3. USO DO PASSO (STEP) NO RANGE
print("\n--- TESTE DE SENSORES (INTERVALO 5s) ---")
for segundo in range(0, 31, 5):
    print(f"Litura realizada aos {segundo} segundos.")

print("\nProcessamento concluído.")

Desafio: Carregamento de Kamehameha

Você deve simular o carregamento de energia. O script deve:

  • Pedir ao utilizador o tempo de carga (em segundos).
  • Usar um for para percorrer esse tempo.
  • A cada segundo, deve imprimir a palavra "KA", "ME", "HA", "ME"... em sequência.
  • Dica: Pode usar o operador de resto % dentro do loop para alternar entre as sílabas ou uma lógica de contagem. No final, imprima um grande "HAAAAA!!!!".
