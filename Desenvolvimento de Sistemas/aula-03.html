Aula 03: Operadores e Expressões

Fundamentos matemáticos para construção de lógica de negócios, cálculos financeiros e processamento de dados.

1. Fundamentos Matemáticos

Aritmética Básica

  • + Adição (Soma valores ou une textos)
  • - Subtração
  • * Multiplicação
  • / Divisão (Sempre retorna decimal float)

Operadores Avançados

  • ** Exponenciação: Potência (Ex: 2³ = 8)
  • // Divisão Inteira: Ignora decimais
  • % Módulo: Resto da divisão (Útil para par/ímpar)

2. Exemplo Prático: Sistema Comercial

sales_system.py

Este exemplo demonstra o cálculo de descontos e médias em um cenário de vendas reais.

# --- SISTEMA DE VENDAS E ESTATÍSTICAS ---

# 1. ENTRADA DE DADOS (Simulação)
produto = "Notebook Gamer"
preco_unitario = 4500.00
quantidade = 3

# 2. CÁLCULO BÁSICO
total_bruto = preco_unitario * quantidade

# 3. CÁLCULO DE DESCONTO (Porcentagem)
# Desconto de 15% para pagamento à vista
fator_desconto = 0.15 
valor_desconto = total_bruto * fator_desconto
total_liquido = total_bruto - valor_desconto

print("--- NOTA FISCAL ---")
print("Produto:", produto)
print("Total Bruto: R$", total_bruto)
print("Desconto Aplicado: R$", valor_desconto)
print("Total a Pagar: R$", total_liquido)

# 4. PARCELAMENTO (Divisão)
parcelas = 12
valor_parcela = total_bruto / parcelas

print("\n--- OPÇÃO PARCELADA ---")
print("12x de R$", valor_parcela)

# 5. PRECEDÊNCIA MATEMÁTICA
# Cálculo de média de vendas da semana
vendas_seg = 1200
vendas_ter = 3400
vendas_qua = 5000

# Uso obrigatório de parênteses para a soma acontecer antes da divisão
media_vendas = (vendas_seg + vendas_ter + vendas_qua) / 3

print("\nMédia de vendas (3 dias): R$", media_vendas)

3. Exercícios de Fixação

Tema: One Piece - Utilize seus conhecimentos matemáticos para resolver os problemas da Grand Line.

Nível: Fácil

EXERCÍCIO 01

Soma de Berries: Crie duas variáveis: recompensa_usopp (500 milhões) e recompensa_franky (394 milhões). Crie uma variável total somando as duas e imprima o resultado.

EXERCÍCIO 02

Gear 5: Luffy aumentou seu poder. Crie uma variável forca_base = 5000. Use o operador de multiplicação * para calcular o dobro dessa força.

EXERCÍCIO 03

O Saquê do Zoro: Zoro tem 100 garrafas e quer dividir igualmente com o Jinbe. Use a divisão / para mostrar quantas garrafas cada um recebe.

Nível: Médio

EXERCÍCIO 04

Média de Força CP9: Três inimigos possuem força: 80, 90 e 75. Some os três valores e divida por 3 para achar a média. Dica: Use parênteses na soma.

EXERCÍCIO 05

Log Pose: A viagem dura 4850 minutos. Use // para achar as horas inteiras e % para achar os minutos restantes.
