Aula 06: Estruturas Condicionais

Dê inteligência ao seu software. Aprenda a criar desvios lógicos onde o programa decide qual caminho seguir com base em condições.

1. Desvios de Lógica

Como o Python avalia perguntas e executa blocos de código específicos.

A Estrutura IF

O if (se) executa um bloco apenas se a condição for Verdadeira. É o filtro inicial de qualquer decisão.

ELIF e ELSE

O elif permite testar múltiplas condições em sequência. O else é o caminho final, executado se nada antes for verdade.

Operadores

Usamos comparadores para criar condições: == (igual), != (diferente), >, <, >= e <=.

2. Exemplo Prático: Análise de Crédito

Um sistema real que decide se um cliente pode ou não obter um empréstimo bancário.

FINANCE_ALGO.PY 
# --- SISTEMA AUTOMATIZADO DE CRÉDITO BANCÁRIO ---

print("=== ANÁLISE DE RISCO v2.0 ===")

# Entrada de dados do cliente
salario = float(input("Informe o salário mensal: R$ "))
idade = int(input("Informe a idade: "))
score_serasa = int(input("Informe o Score de crédito (0-1000): "))

# Lógica de Decisão (Estrutura Condicional)
if salario < 1500:
    print("\nSTATUS: REPROVADO")
    print("Motivo: Renda mínima insuficiente.")

elif score_serasa < 300:
    print("\nSTATUS: EM ANÁLISE HUMANA")
    print("Motivo: Histórico de crédito baixo.")

elif idade < 18:
    print("\nSTATUS: REPROVADO")
    print("Motivo: Cliente menor de idade.")

else:
    print("\nSTATUS: APROVADO COM SUCESSO!")
    limite = salario * 0.3
    print(f"Seu limite inicial será de R$ {limite:.2f}")

print("\nObrigado por utilizar o nosso sistema.")
