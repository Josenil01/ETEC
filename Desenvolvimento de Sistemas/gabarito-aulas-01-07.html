Exercício 3 - Calculadora de operações essenciais

Ler dois inteiros e exibir operações principais.

Resposta

a = int(input("Digite o primeiro número: "))
b = int(input("Digite o segundo número: "))

print("Soma:", a + b)
print("Subtração:", a - b)
print("Multiplicação:", a * b)
print("Divisão real:", a / b)
print("Divisão inteira:", a // b)
print("Resto:", a % b)

Exercício 4 - Média com entrada de dados

Ler 3 notas em float e calcular média.

Resposta

n1 = float(input("Nota 1: "))
n2 = float(input("Nota 2: "))
n3 = float(input("Nota 3: "))

media = (n1 + n2 + n3) / 3
print(f"Média final: {media:.2f}")

Exercício 5 - Sistema de desconto simples

Calcular desconto percentual e valor final.

Resposta

valor = float(input("Valor do produto: R$ "))
percentual = float(input("Desconto (%): "))

desconto = valor * (percentual / 100)
valor_final = valor - desconto

print(f"Desconto: R$ {desconto:.2f}")
print(f"Valor final: R$ {valor_final:.2f}")

Exercício 6 - Tratamento de texto do usuário

Aplicar strip, upper, lower e contar caracteres.

Resposta

nome_completo = input("Digite seu nome completo: ")
nome_limpo = nome_completo.strip()

print("Sem espaços extras:", nome_limpo)
print("Maiúsculas:", nome_limpo.upper())
print("Minúsculas:", nome_limpo.lower())
print("Total de caracteres:", len(nome_limpo))

Exercício 7 - Analisador de frase

Verificar presença de palavra e substituir texto.

Resposta

frase = input("Digite uma frase: ")
palavra = input("Palavra para procurar: ")
nova_palavra = input("Nova palavra para substituir: ")

print("Palavra encontrada?", palavra in frase)
print("Frase atualizada:", frase.replace(palavra, nova_palavra))

Exercício 8 - Classificação por nota (if/elif/else)

Classificar em Aprovado, Recuperação ou Reprovado.

Resposta

nota = float(input("Digite a nota final: "))

if nota >= 7:
    print("Aprovado")
elif nota >= 5:
    print("Recuperação")
else:
    print("Reprovado")

Exercício 9 - Validação de acesso com operadores lógicos

Permitir entrada com regra combinada de idade, documento e acompanhante.

Resposta

idade = int(input("Idade: "))
possui_documento = input("Possui documento? (True/False): ") == "True"
acompanhado = input("Está acompanhado? (True/False): ") == "True"

if (idade >= 18 and possui_documento) or acompanhado:
    print("Entrada permitida")
else:
    print("Entrada negada")

Exercício 10 - Menu com match case

Criar menu de opções e tratar opção inválida com caso padrão.

Resposta

opcao = int(input("Escolha uma opção (1-4): "))

match opcao:
    case 1:
        print("Cadastrar aluno")
    case 2:
        print("Exibir notas")
    case 3:
        print("Atualizar dados")
    case 4:
        print("Sair do sistema")
    case _:
        print("Opção inválida")
