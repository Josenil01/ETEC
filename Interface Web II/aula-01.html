Aula 01: O Caminho do Ninja Dev: Setup e Primeiros Jutsus

Prepare seu ambiente como um verdadeiro Shinobi e aprenda os jutsus fundamentais para se comunicar com o computador.

1. Academia Ninja: Preparando seu Arsenal

Ferramenta Principal: VS Code

O Visual Studio Code é a sua "bolsa de kunais". É um editor de código leve e poderoso onde escreveremos todos os nossos jutsus (códigos).

Pergaminhos (Extensões) Essenciais

  • 01. Live Server: Permite que você veja seu site atualizando em tempo real, como um Jutsu de Transformação instantâneo.
  • 02. Prettier: Mantém seu código organizado e alinhado, deixando seus "selos de mão" sempre perfeitos.

2. Entendendo os Chakras da Web

HTML O esqueleto da página. Define a estrutura, como o corpo de um ninja.
CSS A aparência. Define as cores e estilos, como o uniforme de Konoha.
JavaScript (JS) O Chakra! Dá vida e poder à página, permitindo executar jutsus (interações).

3. Executando seus Primeiros Jutsus

aula01_jutsu_inicial.js 
// --- JUTSU DE COMUNICAÇÃO: console.log() ---
// Mostra mensagens no console do navegador (F12). É o seu rádio ninja.
console.log("Tô certo, dattebayo!");

// --- JUTSU DE INVOCAÇÃO E INTERAÇÃO ---
// Criam pop-ups para interagir com o usuário.

// 1. alert(): Apenas mostra uma mensagem.
alert("Atenção! Missão de Rank S recebida!");

// 2. confirm(): Pergunta Sim/Não e retorna true (OK) ou false (Cancelar).
const aceitou = confirm("Você aceita a missão de resgatar o Sasuke?");
console.log("Missão aceita:", aceitou);

// 3. prompt(): Pede para o usuário digitar algo e retorna o texto.
const nomeNinja = prompt("Qual o seu nome de ninja para o registro?");
alert("Bem-vindo à Aliança Shinobi, " + nomeNinja + "!");

// --- COMO CONECTAR SEU PERGAMINHO DE JUTSUS (arquivo .js) ---
// Para que esses jutsus funcionem, seu arquivo JavaScript precisa ser "invocado" pelo HTML.
// No seu arquivo index.html, adicione esta linha ANTES de fechar a tag </body>:
// <script src="aula01_jutsu_inicial.js"></script>

4. Missões Nível Genin

Nível: Fácil (Interação Básica)

01

Use `prompt` para perguntar "Qual seu jutsu favorito?". Em seguida, use `alert` para exibir "Executando [jutsu] agora!".

02

Use `prompt` para perguntar o nome do usuário. Depois, use `console.log` para saudá-lo: "Seja bem-vindo ao teste Chunnin, [nome]!".

03

Use `confirm` para perguntar "Você prefere Ramen ou Sushi?". Exiba a resposta (`true` ou `false`) no console.

Nível: Médio (Manipulação de Texto)

04

Registro de Time: Use `prompt` 3 vezes para registrar o nome de 3 ninjas. Depois, use um único `console.log` para exibir "Time 7 formado com: [ninja1], [ninja2] e [ninja3]".

05

Relatório para o Hokage: Peça o nome do ninja com `prompt` e a mensagem do relatório com outro `prompt`. Exiba no console: "De: [nome do ninja] | Para: Hokage | Mensagem: [mensagem]".

Nível: Desafio (Jounin)

06

Criador de Cartão Ninja: Use `prompt` para pedir Nome, Vila e Jutsu Principal. Depois, use `alert` para exibir um "cartão" formatado usando `\n` para pular linhas. Ex: "--- CARTÃO NINJA ---\nNome: Naruto\nVila: Folha\nJutsu: Rasengan".
