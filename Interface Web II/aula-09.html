Aula 09: DOM - Entrando na Matrix

Aprenda a alterar a realidade (sua página) e criar novos elementos do zero com o poder do JavaScript.

1. A Pílula Vermelha: Ver para Mudar

Na aula anterior, aprendemos a "ver" a Matrix (selecionar elementos do DOM usando getElementById e querySelector). Agora, vamos tomar a pílula vermelha e aprender a "dobrar as regras".

A manipulação do DOM nos permite alterar o conteúdo, o estilo e a estrutura da nossa página em tempo real. Você não está mais restrito ao HTML estático; com o JavaScript, você pode criar experiências dinâmicas e interativas, fazendo elementos aparecerem, mudarem de cor ou desaparecerem de acordo com a vontade do usuário.

2. O Arsenal de Neo

Modificadores (Alteram o que existe)
.innerText

Altera apenas o texto visível de um elemento. Ignora tags de HTML.

.innerHTML

Altera o conteúdo permitindo injetar novas tags HTML junto com o texto.

.style.propriedade

Altera o CSS diretamente via JavaScript (ex: .style.color = "red").

Criadores (Fazem do zero)
document.createElement('tag')

Cria um novo elemento HTML na "memória" do navegador, mas ainda não o coloca na tela.

.className

Atribui as classes CSS ao elemento recém-criado antes de mostrá-lo.

pai.appendChild(filho)

Pega o elemento que estava na memória e o "planta" dentro de um elemento pai na página real.

Passo 3 Entrando na Matrix (Prática com JS)

Aqui a mágica acontece. Primeiro, alteramos o que já existe (O Título). Depois, criamos elementos inteiramente novos do zero e os anexamos à página.

Desperte, Neo.

Escreva para entender as regras.

aula09_matrix.js
// --- 1. MODIFICANDO ELEMENTOS EXISTENTES ---
const titulo = document.getElementById('titulo');

// Alterando o texto do título
titulo.innerText = 'Você está na Matrix.';
// Alterando o estilo em tempo real
titulo.style.color = '#fff'; 


// --- 2. CRIANDO ELEMENTOS DO ZERO ---
const areaAgentes = document.getElementById('agentes');

// Lista de dados (Imagine que vieram de um banco de dados)
const dadosAgentes = [
    { nome: 'Agente Smith', foto: 'https://i.imgur.com/5J3o76B.png' },
    { nome: 'Agente Jones', foto: 'https://i.imgur.com/8QWv7uR.png' }
];

// Para cada agente na lista, vamos fabricar o HTML!
dadosAgentes.forEach(agente => {
    // 1. CREATE: Cria a div vazia na memória
    const novoCard = document.createElement('div');
    
    // 2. CLASS: Aplica a classe que estilizamos no CSS
    novoCard.className = 'card';
    
    // 3. INNER HTML: Preenche o miolo do card com tags e dados
    novoCard.innerHTML = `
        <img src="${agente.foto}" alt="${agente.nome}">
        <p>${agente.nome}</p>
    `;
    
    // 4. APPEND: Acopla o elemento finalizado na tela real!
    areaAgentes.appendChild(novoCard);
});

6. Treinamento no Dojo

Nível: Potencial (Fácil)

01

Selecione o <h1> com id "titulo". Use a propriedade .innerText para mudar o texto dele para "Olá, Neo.".

02

Selecione a tag <body> da sua página (Dica: document.querySelector('body')) e mude a cor de fundo usando a propriedade .style.backgroundColor para "#0a0a0a".

Nível: O Escolhido (Médio)

03

Déjà vu: Crie um novo parágrafo do zero usando document.createElement('p'). Coloque o texto "Eu já estive aqui antes." dentro dele. Depois, adicione-o ao final do body usando o appendChild.

04

Telefone Tocando: Selecione o <h1>. Em vez de usar `innerText`, mude o .innerHTML dele para "O telefone está tocando... <b>Atenda!</b>". Veja como a tag HTML <b> é interpretada e renderizada!

Nível: Arquiteto (Desafio)

05

Construtor de "Glitches":

1. Use o prompt() do JavaScript para perguntar ao usuário: "Qual a mensagem do glitch?". Guarde isso em uma variável.
2. Crie um novo elemento <p> na memória do JS.
3. Defina o texto deste parágrafo com a mensagem digitada pelo usuário.
4. Adicione uma classe ao parágrafo fazendo: elemento.className = 'glitch'.
5. Vá até o seu arquivo style.css e crie essa classe .glitch (ex: cor vermelha e itálico).
6. Finalmente, adicione (append) o novo parágrafo dentro da <div id="agentes">.
