Aula 09: Manipulando o DOM - Entrando na Matrix

Aprenda a alterar a realidade (sua página) e criar novos elementos do zero com o poder do JavaScript.

1. A Pílula Vermelha: Ver para Mudar

Na aula anterior, aprendemos a "ver" a Matrix (selecionar elementos do DOM). Agora, vamos tomar a pílula vermelha e aprender a "dobrar as regras". A manipulação do DOM nos permite alterar o conteúdo, o estilo e a estrutura da nossa página em tempo real, criando experiências dinâmicas e interativas para o usuário.

2. O Arsenal de Neo

ModificadoresPropriedades como `.innerText`, `.innerHTML` e `.style` que nos permitem alterar o conteúdo e a aparência dos elementos.
CriadoresMétodos como `document.createElement()` e `elemento.appendChild()` para criar novos elementos do zero e inseri-los na página.

5. Entrando na Matrix (Prática com JS)

Siga o Coelho Branco.

Não há colher... nem Ctrl+C. Digite o código para ver o quão fundo a toca do coelho vai.

aula09_matrix.js
// --- 1. Alterando a Realidade (Modificando Elementos) ---
const titulo = document.getElementById('titulo');
titulo.innerText = "Você está na Matrix.";
titulo.style.color = "#00ff00";

// --- 2. Criando a Realidade (Criando e Inserindo Elementos) ---
const agentes = [
  { nome: "Agente Smith", imagem: "https://via.placeholder.com/100/050505/00ff00?text=Smith" },
  { nome: "Agente Brown", imagem: "https://via.placeholder.com/100/050505/00ff00?text=Brown" },
  { nome: "Agente Jones", imagem: "https://via.placeholder.com/100/050505/00ff00?text=Jones" }
];

const containerAgentes = document.getElementById('agentes');

agentes.forEach(agente => {
  const card = document.createElement('div');
  const img = document.createElement('img');
  const nomeAgente = document.createElement('h3');

  card.className = 'card';
  img.src = agente.imagem;
  nomeAgente.innerText = agente.nome;

  card.appendChild(img);
  card.appendChild(nomeAgente);

  containerAgentes.appendChild(card);
});

6. Treinamento no Dojo

Nível: Potencial (Fácil)

01

Selecione o `<h1>` com id "titulo" e mude seu texto para "Olá, Neo.".

02

Selecione o `body` da página e mude sua cor de fundo (`backgroundColor`) para `#0a0a0a`.

Nível: O Escolhido (Médio)

03

Déjà vu: Crie um novo parágrafo `<p>` com o texto "Eu já estive aqui antes.". Adicione-o ao final do `body`.

04

Telefone Tocando: Selecione o `<h1>` e mude seu `innerHTML` para "O telefone está tocando... <b>Atenda!</b>".

Nível: Arquiteto (Desafio)

05

Construtor de "Glitches":
1. Use `prompt` para perguntar "Qual a mensagem do glitch?".
2. Crie um novo elemento `<p>`.
3. Defina o texto do novo parágrafo com a mensagem do usuário.
4. Adicione a classe "glitch" ao parágrafo (`elemento.className = 'glitch'`).
5. No seu CSS, crie uma classe `.glitch` com `color: red;` e `font-style: italic;`.
6. Adicione o novo parágrafo ao `body` da página.
