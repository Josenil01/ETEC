Aula 03: Operadores e Expressões Matemáticas

Aprenda a calcular tesouros, comparar recompensas e tomar decisões como um verdadeiro Rei dos Piratas usando os operadores do JavaScript.

1. A Grande Rota: Navegando com Operadores

Assim como um navegador usa bússolas e mapas, um programador usa **operadores** para manipular dados. Eles são os símbolos que nos permitem fazer contas, comparar informações e tomar decisões lógicas, nos guiando através dos mares perigosos da programação.

2. O Haki dos Operadores

Aritméticos (+, -, *, /) O básico para todo pirata: somar, subtrair, multiplicar e dividir o tesouro.
Comparação (>, <, ===) Para saber se sua recompensa é maior que a do Buggy, ou se o tesouro é o verdadeiro One Piece.
Lógicos (&&, ||, !) Para tomar decisões: você precisa de um bom navio **E (&&)** uma boa tripulação para ir à Grand Line.

// --- 1. Calculando Recompensas (Operadores Aritméticos) ---
const recompensaLuffy = 3000000000;
const recompensaZoro = 1111000000;

const recompensaTotal = recompensaLuffy + recompensaZoro;
console.log("Recompensa combinada do Capitão e Imediato:", recompensaTotal);

// --- 2. Comparando Forças (Operadores de Comparação) ---
const recompensaBuggy = 3189000000;
const luffyMaisForte = recompensaLuffy > recompensaBuggy; // Isso será false

console.log(`A recompensa do Luffy é maior que a do Buggy? ${luffyMaisForte}`);

const nomeDoTesouro = "One Piece";
console.log("Este é o tesouro verdadeiro?", nomeDoTesouro === "One Piece"); // true

// --- 3. Decisões de Capitão (Operadores Lógicos) ---
const temComida = true;
const temNavegador = true;
const marinhaPorPerto = false;

// Para navegar, precisa de comida E um navegador
const podeNavegar = temComida && temNavegador;
console.log(`Podemos zarpar para a próxima ilha? ${podeNavegar}`);

// Para dar uma festa, precisa de comida OU o Luffy estar feliz
const podeFestejar = temComida || recompensaLuffy > 0;
console.log(`Bora ter um banquete? ${podeFestejar}`);

// Estamos seguros se NÃO (!) tiver a Marinha por perto
const estamosSeguros = !marinhaPorPerto;
console.log(`A costa está limpa? ${estamosSeguros}`);

4. Desafios do Novo Mundo

Nível: Marinheiro de Primeira Viagem (Fácil)

01

Declare duas recompensas (ex: Sanji e Usopp) e some-as. Exiba o resultado no console.

02

Crie uma variável `minhaRecompensa` com valor 50000. Verifique se ela é maior que a recompensa de Chopper (1000) e exiba o resultado (`true` ou `false`).

03

Crie duas variáveis booleanas: `encontrouPoneglyph = true` e `robinEstaPresente = true`. Verifique se as duas são verdadeiras usando `&&`.

Nível: Capitão Pirata (Médio)

04

Divisão do Tesouro: Crie uma variável com um valor de tesouro (ex: 100000 Berries) e outra com o número de tripulantes (ex: 9). Calcule e exiba no console quanto cada um recebe.

05

Aliança Pirata: Use `prompt` para perguntar a recompensa de um pirata aliado. Use `confirm` para perguntar se ele é confiável. Exiba `true` no console apenas se a recompensa for maior que 100,000,000 **E** ele for confiável.

Nível: Yonkou (Desafio)

06

Simulador de Batalha Naval:
1. Use `prompt` para perguntar a força do navio inimigo (um número de 1 a 10).
2. Use `prompt` para perguntar o número de tripulantes inimigos.
3. Use `confirm` para perguntar se o mar está calmo.
4. A condição para vitória é: (Nossa força, que é 8, for maior que a do inimigo **E** o número de inimigos for menor que 50) **OU** o mar estiver calmo. Crie uma variável `podemosVencer` com essa lógica e exiba o resultado no console.
