Aula 12: Funções - A Forja dos Anões

Aprenda a forjar blocos de código reutilizáveis (funções) para criar ferramentas poderosas e evitar a repetição.

1. A Grande Forja de Erebor

Um mestre ferreiro anão não constrói uma espada do zero toda vez. Ele tem um processo, uma "receita". Em programação, essa receita é uma **função**. Funções são blocos de código que podemos nomear e reutilizar quantas vezes quisermos. Em vez de escrever o mesmo código para somar números em dez lugares diferentes, criamos uma única função `somar()` e a chamamos onde for necessário. Isso torna nosso código mais limpo, organizado e fácil de manter.

2. As Ferramentas do Ferreiro

Declaração A "planta" da sua ferramenta. Usamos a palavra-chave `function` seguida de um nome, parênteses `()` e chaves `{}`.
Parâmetros As "matérias-primas" que sua função recebe para trabalhar. Ficam dentro dos parênteses.
Return (Retorno) O "produto final" que a função devolve após terminar seu trabalho. Uma função pode ou não retornar um valor.

5. Forjando as Ferramentas (JS)

Segredo de Ferreiro!

Copiar a planta não te torna um mestre. Forje cada linha de código para dominar a arte das funções.

aula12_forja.js
// --- 1. Selecionando os materiais da bancada ---
const inputNum1 = document.getElementById('num1');
const inputNum2 = document.getElementById('num2');
const btnSomar = document.getElementById('btn-somar');
const btnSubtrair = document.getElementById('btn-subtrair');
const pResultado = document.getElementById('resultado');

// --- 2. Forjando as ferramentas (Declarando as Funções) ---

// Função para somar dois números
function somar(a, b) {
  return a + b;
}

// Função para subtrair dois números
function subtrair(a, b) {
  return a - b;
}

// --- 3. Usando as ferramentas (Adicionando Eventos) ---
btnSomar.addEventListener('click', () => {
  // Pega os valores dos inputs e converte para número
  const num1 = parseFloat(inputNum1.value);
  const num2 = parseFloat(inputNum2.value);

  // Chama a função e guarda o resultado
  const resultado = somar(num1, num2);

  // Mostra o resultado na tela
  pResultado.innerText = resultado;
});

btnSubtrair.addEventListener('click', () => {
  const num1 = parseFloat(inputNum1.value);
  const num2 = parseFloat(inputNum2.value);
  const resultado = subtrair(num1, num2);
  pResultado.innerText = resultado;
});

6. As Provas da Guilda dos Ferreiros

Nível: Aprendiz

01

Crie uma função `saudacao` que receba um nome como parâmetro e exiba no console "Saudações, [nome]!".

02

Crie uma função `dobrar` que receba um número e retorne o dobro dele.

Nível: Ferreiro

03

Forjando uma Espada: Crie uma função `forjar` que recebe dois parâmetros: `metal` e `qualidade`. A função deve retornar a string: "Espada de [metal] forjada com qualidade [qualidade]!".

04

Calculadora de Área: Crie uma função `calcularArea` que receba `largura` e `altura` e retorne a área. Conecte-a a inputs e um botão para exibir o resultado na tela.

Nível: Mestre da Forja

05

Calculadora de Obra-Prima: Melhore a calculadora da aula. Crie uma única função `calcular(num1, num2, operacao)`. A `operacao` será uma string ('somar', 'subtrair', etc.). Dentro da função, use `if/else if` ou `switch` para decidir qual cálculo fazer e retornar o resultado. Adapte os botões para chamar essa única função com a operação correta.
