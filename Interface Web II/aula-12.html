Aula 12: Assincronia e APIs

Aprenda a usar o fetch para buscar dados em servidores externos através de eventos, sem congelar a sua página.

1. A Arma de Portais (Assincronia)

Na web, frequentemente precisamos de buscar informações que estão guardadas num servidor distante (como uma base de dados de filmes, meteorologia, ou personagens). Este processo demora tempo (milissegundos ou segundos).

Se o JavaScript simplesmente parasse e esperasse pela resposta, a sua página ficaria completamente "congelada". É aqui que entra a Assincronia. O JS atira a Arma de Portais (faz o pedido) e continua a trabalhar noutras coisas. Quando os dados chegam da outra dimensão, ele processa-os e mostra-os no ecrã.

2. Ferramentas Interdimensionais

A Comunicação
API (Interface)

O "empregado de mesa" do servidor. Nós fazemos um pedido (ex: "Quero a personagem 5"), a API vai à cozinha (base de dados) e traz-nos o prato pronto.

JSON

A língua universal da web. Os dados viajam pela internet em formato de texto estruturado que se parece muito com um Objeto do JavaScript ({"nome": "Morty", "status": "Vivo"}).

Os Comandos JS
fetch("url")

A função que faz o pedido para uma URL externa. Ela devolve uma Promise (uma promessa de que os dados chegarão no futuro).

async / await

Colocamos async antes da nossa função para avisar que haverá um atraso. Colocamos await antes do fetch para dizer ao código: "Pausa esta linha específica e só continua quando a resposta chegar".

Passo 3 O Evento e a Função Async (JS)

Tudo se une aqui. Ouvimos o clique, ativamos o loader, fazemos o fetch na API usando await, convertemos para JSON e usamos o DOM para injetar o resultado!

portal.js
// 1. Selecionamos os elementos do HTML
const btnPortal = document.getElementById('btn-portal');
const loader = document.getElementById('loader');
const areaResultado = document.getElementById('resultado-api');

// 2. Criamos o evento 'click'. A função DEVE ter 'async' para usarmos 'await'
btnPortal.addEventListener('click', async () => {
    
    // PASSO A: Ocultar o conteúdo antigo e mostrar a animação de carregamento
    areaResultado.innerHTML = ""; 
    loader.classList.remove('hidden');

    // Usamos try/catch para capturar falhas de internet ou de API
    try {
        // Sorteia um ID de 1 a 826 (limite da API do Rick and Morty)
        const idAleatorio = Math.floor(Math.random() * 826) + 1;
        
        // PASSO B: Disparar o 'fetch' e ESPERAR (await) a resposta chegar
        const resposta = await fetch(`https://rickandmortyapi.com/api/character/${idAleatorio}`);
        
        // PASSO C: Converter a resposta da internet para JSON (Objeto JS)
        const dados = await resposta.json();

        // PASSO D: Ocultar o carregamento
        loader.classList.add('hidden');

        // PASSO E: Manipulação do DOM (O que aprendemos na Aula 09)
        areaResultado.innerHTML = `
            <div class="char-card">
                <img src="${dados.image}" alt="${dados.name}">
                <div class="char-info">
                    <h3 class="font-bold text-xl text-lime-400">${dados.name}</h3>
                    <p class="text-sm text-slate-300">Status: ${dados.status}</p>
                    <p class="text-sm text-slate-300">Espécie: ${dados.species}</p>
                </div>
            </div>
        `;

    } catch (erro) {
        // Se algo falhar (ex: sem net), cai aqui!
        loader.classList.add('hidden');
        areaResultado.innerHTML = '<p class="text-red-500">Erro no portal. Tente novamente!</p>';
        console.error(erro);
    }
});

4. Desafios Multiversais (Prática)

Nível: Morty (Fácil)

01

Primeiro Contacto: Crie uma função async simples ligada a um botão. Use const resposta = await fetch('https://rickandmortyapi.com/api/character/1');, depois const dados = await resposta.json(); e use um console.log(dados) para inspecionar os dados do próprio Rick Sanchez na consola do DevTools.

Nível: Summer (Médio)

02

Tratamento de Anomalias: Modifique o código do desafio anterior envolvendo-o num bloco try { ... } catch (erro) { ... }. Para testar o erro, introduza uma URL falsa no fetch (ex: fetch('https://fake-url-errada.com')) e faça com que o catch use um alert('O servidor explodiu!').

Nível: Rick Sanchez (Desafio)

03

Listagem de Dimensões (Juntando tudo!):

A URL https://rickandmortyapi.com/api/location retorna uma lista de planetas dentro da propriedade dados.results (que é um Array).
1. Crie um botão "Buscar Planetas" e uma <ul> vazia.
2. Faça o fetch na URL acima e converta para JSON.
3. Use um laço forEach no array dados.results.
4. Para cada planeta, use document.createElement('li'), coloque o .name do planeta dentro dele, e injete na sua lista do HTML!
