Aula 04: Funções e Escopo no Aranhaverso

Aprenda a criar seus próprios "lançadores de teia" (funções) para reutilizar poderes e entenda onde cada herói (variável) pode atuar (escopo).

1. Com Grandes Poderes, Vêm Grandes Funções

No Aranhaverso, cada herói tem seu jeito de lançar teias, mas a ação é a mesma. Em programação, se você precisa repetir uma mesma tarefa várias vezes, você não escreve o código de novo. Você cria uma **função**: um bloco de código reutilizável que pode ser "chamado" a qualquer momento para executar uma ação específica. É o seu lançador de teias pessoal!

2. A Ciência por Trás da Teia

Função Um bloco de código nomeado e reutilizável. Você define uma vez e usa quantas vezes quiser.
Parâmetros As "configurações" da sua função. Ex: o alvo da teia, a força do pulo, etc.
Escopo A "dimensão" onde uma variável existe. Se criada dentro de uma função, só existe lá dentro (escopo local).

3. Através do Aranhaverso (Prática)

Sentido Aranha Ativado!

Este código está protegido! Para dominar seus poderes, você precisa praticar, não copiar. Digite cada linha, jovem herói!

aula04_aranhaverso.js 
// --- 1. Função Simples (Sem Parâmetros) ---
function saudacaoAranha() {
  console.log("E aí, Amigo da Vizinhança!");
}

saudacaoAranha(); // "Chamando" a função para ser executada

// --- 2. Função com Parâmetros ---
// 'alvo' é um parâmetro: uma variável que só existe dentro da função.
function lancarTeia(alvo) {
  console.log(`Lançando teia no ${alvo}!`);
}

lancarTeia("Duende Verde"); // "Duende Verde" é o argumento passado para o parâmetro 'alvo'
lancarTeia("prédio mais próximo");

// --- 3. Função com Retorno (return) ---
// 'return' envia um valor de volta para quem chamou a função.
function calcularDanoDoVenom(forca) {
  const danoTotal = forca * 1.5;
  return danoTotal;
}

const danoCausado = calcularDanoDoVenom(100);
console.log(`O ataque de Venom causou ${danoCausado} de dano!`);

// --- 4. Escopo Global vs. Local ---
const identidadeSecreta = "Peter Parker"; // Escopo Global: acessível em qualquer lugar

function mostrarPensamento() {
  const pensamento = "Preciso pagar o aluguel."; // Escopo Local: só existe aqui dentro
  console.log(`Dentro da função, posso ver a identidade: ${identidadeSecreta}`);
  console.log(`E também o pensamento: ${pensamento}`);
}

mostrarPensamento();

console.log(`Fora da função, ainda vejo a identidade: ${identidadeSecreta}`);
// console.log(pensamento); // Isso daria um ERRO! 'pensamento' não existe aqui fora.

4. Desafios do Multiverso

Nível: Amigo da Vizinhança (Fácil)

01

Crie uma função chamada `fraseDeEfeito` que exiba no console a frase "Com grandes poderes, vêm grandes responsabilidades."

02

Crie uma função `cumprimentar` que receba um `nome` como parâmetro e exiba "Olá, [nome]!".

03

Crie uma função `numeroDoUniverso` que simplesmente `return` o número `1610` (Universo do Miles).

Nível: Herói de Nova York (Médio)

04

Calculadora de Salto: Crie uma função `calcularAlturaDoPulo` que recebe `forcaDoPulo` e `gravidade` como parâmetros. A função deve retornar o resultado de `forcaDoPulo / gravidade`. Chame a função e exiba o resultado no console.

05

Gerador de Nome de Herói: Crie uma função `gerarNomeAranha` que recebe `nome` e `sobrenome`. A função deve retornar uma string no formato "Aranha-[Primeira Letra do Nome][Primeira Letra do Sobrenome]". Ex: `gerarNomeAranha("Peter", "Parker")` deve retornar "Aranha-PP".

Nível: Aranha Lendária (Desafio)

06

Simulador de Portal Multiversal:
1. Crie uma função `ativarGoHomeMachine`.
2. Dentro dela, use `prompt` para perguntar "Qual o universo de destino? (Ex: 928)".
3. Use `confirm` para perguntar "É uma emergência?".
4. A função deve `return` uma string. Se for uma emergência, retorne "EMERGÊNCIA: Abrindo portal para o universo [número]!". Se não, retorne "Viagem agendada para o universo [número].".
5. Chame a função e exiba o resultado retornado em um `alert`.
