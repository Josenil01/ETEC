Aula 17: Migalhas Digitais

Como João e Maria no Mundo Invertido. Aprenda a usar Cookies e o LocalStorage para fazer o navegador lembrar-se de quem você é.

1. A Amnésia da Web

O protocolo HTTP, que estudámos nas aulas passadas, é o que chamamos de um protocolo "stateless" (sem estado). Isto significa que ele sofre de amnésia severa: cada vez que você atualiza a página ou clica num link, o navegador esquece completamente quem você é e o que acabou de fazer.

Mas se a web é amnésica, como é que o Facebook o mantém logado? Como é que a Amazon guarda os produtos no seu carrinho de compras mesmo se fechar o site? Para sobreviver no Mundo Invertido, precisamos de deixar "migalhas digitais" (como no conto de João e Maria) guardadas no próprio navegador.

2. O Kit de Sobrevivência

1. LocalStorage

A mochila pesada. Os dados guardados aqui ficam no navegador para sempre (ou até serem limpos manualmente). Tem grande capacidade de armazenamento (5MB a 10MB).

localStorage.setItem('chave', 'valor') localStorage.getItem('chave')
2. A Magia do JSON & Apagar Rastros

O LocalStorage só guarda texto puro. Se tentar guardar um Objeto {nome: "Will"}, ele vai quebrar. Para gravar, transforme-o em texto com JSON.stringify(obj). Para ler de volta, use JSON.parse(texto).

Para fazer "Logout" ou limpar as migalhas, use localStorage.removeItem('chave').

3. SessionStorage

A memória a curto prazo. Funciona exatamente igual ao LocalStorage, mas apaga-se automaticamente assim que o utilizador fecha o separador (aba) ou o navegador por completo.

sessionStorage.setItem('key', 'val')
4. Cookies (O Superpoder)

As verdadeiras migalhas clássicas. Têm pouco espaço (4KB), mas possuem um superpoder vital: viajam silenciosamente e automaticamente anexados a cada pedido HTTP (`fetch`) para o Servidor. É assim que o servidor sabe que você está logado!

document.cookie = "tema=dark; max-age=86400"
🕵️‍♂️

A Lupa de Investigação (DevTools)

Não codifique às cegas! No seu navegador, prima F12, vá à aba Application (ou Armazenamento) e procure por Local Storage. Você verá uma tabela em tempo real com todas as chaves e valores que o seu JavaScript está a gravar na sua máquina. Ver para crer!

A Engenharia do Portal

HTML O Contentor da Cidade

Temos uma estrutura simples. O segredo está no id="hawkins-container", onde o JavaScript irá injetar a classe sombria quando lermos as migalhas digitais, e no botão btn-limpar.

index.html
<!-- A classe 'upside-down' NÃO está aqui inicialmente -->
<div id="hawkins-container" class="hawkins-town">
    <div class="content-wrapper">
        <h4 id="status-texto">Mundo Normal</h4>
        <p id="ultima-hora"></p>
        <button id="btn-portal" class="hawkins-btn">Abrir Portal</button>
    </div>
</div>

<button id="btn-limpar">Apagar Rastros</button>

CSS As Duas Dimensões

No ficheiro style.css, preparamos as duas classes. A classe .upside-down modifica o estilo do contentor, incluindo uma rotação (opcional).

style.css
/* O Mundo Normal */
.hawkins-town {
    background-color: #f0fdf4; 
    border: 2px solid #166534;
    color: #14532d;
    padding: 2rem;
    transition: all 0.8s ease-in-out; 
}

/* A classe ativada pelo LocalStorage */
.upside-down {
    background-color: #0c0a09;
    border-color: #991b1b;
    color: #fca5a5;
    transform: rotate(180deg);
}

/* Evitar que as letras fiquem de pernas para o ar */
.upside-down .content-wrapper {
    transform: rotate(-180deg);
}

JS Escrevendo e Lendo a Memória (JSON)

No ficheiro script.js fazemos 3 operações: ler a memória ao iniciar a página (usando JSON.parse), gravar um Objeto na memória ao clicar (usando JSON.stringify), e apagar os rastros com o botão Limpar (usando removeItem).

script.js
document.addEventListener('DOMContentLoaded', () => {
    const container = document.getElementById('hawkins-container');
    const btnPortal = document.getElementById('btn-portal');
    const statusTexto = document.getElementById('status-texto');
    const ultimaHora = document.getElementById('ultima-hora');
    const btnLimpar = document.getElementById('btn-limpar');

    // --- 1. A LEITURA (Ao carregar a página) ---
    // LocalStorage guarda apenas texto!
    const dadosEmTexto = localStorage.getItem('hawkins_estado');

    if (dadosEmTexto) {
        // Transforma o texto de volta para um Objeto JS real
        const estadoGuardado = JSON.parse(dadosEmTexto);
        
        if (estadoGuardado.dimensao === 'invertida') {
            container.classList.add('upside-down');
            statusTexto.innerText = 'Mundo Invertido';
            btnPortal.innerText = 'Fechar Portal';
            ultimaHora.innerText = `Aberto às: ${estadoGuardado.hora}`;
        }
    }

    // --- 2. A GRAVAÇÃO (Ao clicar no portal) ---
    btnPortal.addEventListener('click', () => {
        container.classList.toggle('upside-down');
        
        // Criamos um Objeto complexo para gravar
        const novoEstado = {
            dimensao: container.classList.contains('upside-down') ? 'invertida' : 'normal',
            hora: new Date().toLocaleTimeString()
        };

        // Gravamos convertendo o Objeto para String!
        localStorage.setItem('hawkins_estado', JSON.stringify(novoEstado));

        // Atualiza interface...
        if (novoEstado.dimensao === 'invertida') {
            statusTexto.innerText = 'Mundo Invertido';
            btnPortal.innerText = 'Fechar Portal';
            ultimaHora.innerText = `Aberto às: ${novoEstado.hora}`;
        } else {
            statusTexto.innerText = 'Mundo Normal';
            btnPortal.innerText = 'Abrir Portal';
            ultimaHora.innerText = '';
        }
    });

    // --- 3. APAGANDO RASTROS (Clear / Logout) ---
    btnLimpar.addEventListener('click', () => {
        localStorage.removeItem('hawkins_estado');
        // Força o recarregamento da página para mostrar que a memória foi limpa
        window.location.reload(); 
    });
});

Atualize a página e force o texto a carregar o valor guardado. Nível: Eleven (Desafio)

03

O Inventário de Objetos (JSON): Crie um objeto em JavaScript: const inventario = { arma: "taco com pregos", quantidade: 1 }.

Use JSON.stringify() para guardar o inventário no LocalStorage na chave "bolso". Depois, recupere o valor do "bolso", use JSON.parse() para o transformar de volta num objeto, e imprima a "arma" num console.log(). É assim que guardamos carrinhos de compras em sites de e-commerce!
