Aula 08: DOM - Invertendo o Mundo dos Elementos

Aprenda a usar o Document Object Model (DOM) para encontrar e manipular elementos da sua página, como se estivesse se comunicando com o Mundo Invertido.

1. O Portal para o Mundo Invertido

Pense na sua página HTML como a cidade de Hawkins. O que você vê no navegador é o "mundo real". O **DOM (Document Object Model)** é como o Mundo Invertido: uma representação exata da sua página, mas em formato de objetos que o JavaScript consegue entender e manipular. Para mudar uma lâmpada (um elemento `

`) em Hawkins, precisamos primeiro encontrá-la no Mundo Invertido (no DOM).

2. Ferramentas de Exploração

document Um objeto global que representa toda a sua página HTML. É o portão de entrada para o DOM.
Seletores Os "walkie-talkies" que usamos para chamar elementos específicos: por ID, por classe, por tag, etc.

5. Em Hawkins (Prática com JavaScript)

Interferência do Mundo Invertido!

Não dá pra copiar o sinal. Digite o código para se comunicar com os elementos da página e salvá-los do Demogorgon!

aula08_hawkins.js 
// O código abaixo irá interagir com o HTML da seção "O Cenário".

// --- 1. Encontrando o laboratório pelo ID ---
const laboratorio = document.getElementById('laboratorio-hawkins');
console.log(laboratorio);

// --- 2. Encontrando os monstros pela CLASSE ---
const monstros = document.getElementsByClassName('monstro');
console.log(`Monstros encontrados: ${monstros.length}`);

// --- 3. Encontrando todos os parágrafos pela TAG ---
const paragrafos = document.getElementsByTagName('p');
console.log(`Total de parágrafos na página: ${paragrafos.length}`);

// --- 4. O método moderno: querySelector ---
const primeiroMonstro = document.querySelector('.monstro');
const mensagemJoyce = document.querySelector('#mensagem-joyce');
console.log(`A Joyce está dizendo: "${mensagemJoyce.innerText}"`);

// --- 5. O mais poderoso: querySelectorAll ---
const todosOsMonstros = document.querySelectorAll('.monstro');
todosOsMonstros.forEach(monstro => {
  console.log(`Perigo: ${monstro.innerText} está à solta!`);
});
