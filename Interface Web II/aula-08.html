Aula 08: DOM - Invertendo o Mundo

Aprenda a usar o Document Object Model (DOM) para encontrar e manipular elementos da sua página, como se estivesse se comunicando com o Mundo Invertido.

1. O Portal para o Mundo Invertido

Pense na sua página HTML como a cidade de Hawkins. O que você vê no navegador é o "mundo real" (a interface gráfica). O DOM (Document Object Model) é como o Mundo Invertido: uma representação estruturada e exata da sua página, mas em formato de objetos que o JavaScript consegue entender, ler e manipular.

Para mudar uma lâmpada (um elemento <div> ou <p>) em Hawkins, precisamos primeiro encontrá-la no Mundo Invertido usando Seletores, para então alterarmos suas propriedades via código.

2. Ferramentas de Exploração

document Um objeto global que representa toda a sua página HTML. É o portão de entrada obrigatório para acessar o DOM.
getElementById() O walkie-talkie direto. Busca um único elemento específico na página através do seu atributo id="".
querySelector() A bússola moderna. Encontra o primeiro elemento que corresponda a um seletor CSS (ex: .classe ou #id).
querySelectorAll() O radar completo. Captura todos os elementos que correspondam a um seletor CSS e devolve uma lista (NodeList).

Passo 3 Em Hawkins (Prática com JS) - getElementById

Vamos usar getElementById para encontrar a mensagem da Joyce na parede e alterá-la dinamicamente respondendo de onde o Will está.

Interferência Psíquica!

aula08_hawkins.js (Parte 1)
// 1. Selecionando um elemento único pelo ID
const mensagem = document.getElementById('mensagem-joyce');

// 2. Manipulando a propriedade de texto do elemento encontrado
mensagem.innerText = 'ESTOU AQUI';

// Bônus: Manipulando o CSS (style) direto pelo JavaScript
mensagem.style.color = '#eab308'; // Amarelo das luzes de natal

Passo 4 querySelector & querySelectorAll

Agora vamos buscar elementos usando a sintaxe do CSS. O querySelector pega o primeiro que achar, e o querySelectorAll pega todos em uma lista!

Cuidado com o Demogorgon!

Não copie, treine seus dedos!

aula08_hawkins.js (Parte 2)
// 3. querySelector: Acha apenas o PRIMEIRO elemento que bater com o seletor
const laboratorio = document.querySelector('#laboratorio-hawkins');
laboratorio.style.backgroundColor = 'rgba(20, 0, 0, 0.5)'; // Escurece a sala

// 4. querySelectorAll: Acha TODOS os elementos e retorna uma lista
const membrosClube = document.querySelectorAll('.membro');

// Como é uma lista, precisamos usar um laço (forEach) para mexer em todos
membrosClube.forEach((membro) => {
    membro.innerText += ' (Presente)';
    membro.style.color = '#818cf8'; // Azul
});

Nível: Eleven (Desafio)

05

Contato Psíquico:

1. Modifique o HTML original para adicionar 3 parágrafos vazios com os IDs "will", "mike" e "dustin".

4. Se o elemento for encontrado (faça um teste if), use .innerText para mudar o texto dele para "Estou aqui!".
5. Se não, use um alert dizendo "Contato não encontrado no Mundo Invertido."
