Aula 02: Variáveis, Constantes e Tipos de Dados

Domine as "Joias do Infinito" do JavaScript para armazenar e manipular as informações dos heróis mais poderosos da Terra.

1. A Missão: Entendendo as Variáveis

No universo da programação, precisamos guardar informações o tempo todo: o nome de um herói, a quantidade de Joias do Infinito, se o Hulk está calmo ou não. As **variáveis** são como as "fichas de cadastro" da S.H.I.E.L.D.: etiquetas que usamos para nomear e armazenar dados na memória do computador. Sem elas, toda informação se perderia no espaço!

2. O Arsenal: let, const e Tipos de Dados

let Use para valores que podem mudar. Pense no traje do Homem de Ferro: sempre evoluindo.
const Use para valores que nunca mudam. É como o escudo do Capitão América: inquebrável e constante.
Tipos de Dados A "classe" de cada informação. As principais são Texto (String), Número (Number) e Lógico (Boolean).

3. Vingadores, Avante! (Prática Interativa)

aula02_vingadores.js 
// --- Parte 1: Declarando as Fichas dos Heróis ---

let localizacaoAtual = "Torre dos Vingadores";
console.log("Equipe reportando de:", localizacaoAtual);

localizacaoAtual = "Wakanda"; // O valor de 'let' pode mudar.
console.log("Nova localização:", localizacaoAtual);

const lider = "Capitão América"; // O valor de 'const' é fixo.
// lider = "Homem de Ferro"; // Isso quebraria o código!

// --- Parte 2: Protocolo de Recrutamento Interativo ---
alert("Iniciando Protocolo de Recrutamento de Novos Vingadores...");

// Capturando dados do novo recruta com prompt()
const nomeRecruta = prompt("Qual o seu codinome de herói?");
const poderRecruta = prompt("Descreva seu poder principal:");

// Fazendo uma pergunta de Sim/Não com confirm()
const aceitaTermos = confirm("Você jura proteger a Terra e seguir as ordens da S.H.I.E.L.D.?");

// Exibindo a ficha completa no console usando Template Strings
console.log(`
  --- FICHA DE RECRUTAMENTO ---
  Codinome: ${nomeRecruta}
  Poder: ${poderRecruta}
  Aceitou os termos: ${aceitaTermos}
  ---------------------------
`);

alert("Recruta " + nomeRecruta + ", seus dados foram enviados para Nick Fury. Aguarde.");

4. Treinamento na Sala de Perigo

Nível: Recruta (Fácil)

01

Declare uma variável `let` chamada `meuVingador` e atribua a ela o nome do seu Vingador favorito.

02

Declare uma constante `const` chamada `joiasDoInfinito` e atribua a ela o número `6`.

03

Crie uma constante `const` chamada `hulkEsmaga` e atribua a ela o valor booleano `true`.

Nível: Agente (Médio)

04

Apresentação: Crie duas variáveis, `nomeAgente` e `nivelAgente`. Use `console.log` e Template Strings para exibir a frase: "Agente [nome] tem nível de acesso [nível]".

05

Erro de Constante: Tente criar uma `const` para a identidade do Homem-Aranha e depois alterá-la. O que acontece no console? (O objetivo é ver o erro!).

Nível: Vingador (Desafio)

06

Analisador de Nível de Ameaça: Crie um programa que:
1. Pergunta o nome de um personagem com `prompt`.
2. Pergunta o planeta de origem com `prompt`.
3. Pergunta com `confirm` se o personagem é um vilão.
4. Exibe um `alert` final. Se for vilão, o alerta deve ser: "ALERTA VERMELHO: [Nome] de [Planeta] é uma ameaça!". Se não for, deve ser: "Relatório: [Nome] de [Planeta] é um aliado potencial."
