Aula 06: Estruturas Condicionais - O Chapéu Seletor

Aprenda a usar `if`, `else` e `else if` para que seu código tome decisões e escolha a casa (caminho) certa a seguir.

1. A Seleção das Casas

Em Hogwarts, o Chapéu Seletor olha a mente de um bruxo e, baseado em suas qualidades, decide sua casa. Na programação, fazemos o mesmo com **estruturas condicionais**. O nosso código pode "olhar" para uma variável e, dependendo do seu valor (`true` ou `false`), tomar um caminho diferente. É a magia da tomada de decisões!

2. Os Feitiços da Decisão

if (Se) Executa um bloco de código apenas **SE** uma condição for verdadeira. "SE for corajoso, vá para a Grifinória".
else (Senão) Executa um bloco de código alternativo se a condição do `if` for falsa. É o plano B do seu feitiço.
else if (Senão se) Permite testar uma nova condição se a anterior for falsa. Perfeito para as múltiplas qualidades que o Chapéu Seletor avalia.

3. No Salão Principal (Prática)

Feitiço *Protego* Ativado!

Para aprender a magia, um bruxo deve praticar, não copiar. Digite o código para dominar o feitiço!

aula06_hogwarts.js 
// --- 1. Decisão Simples: Acesso à Sala Comunal da Grifinória ---
const senhaDigitada = prompt("Qual a senha?");
const senhaCorreta = "Cabeça de Dragão";

if (senhaDigitada === senhaCorreta) {
  alert("Bem-vindo à Sala Comunal da Grifinória!");
} else {
  alert("Senha incorreta! O retrato da Mulher Gorda não se moverá.");
}

// --- 2. O Chapéu Seletor: if, else if, else ---
const qualidade = prompt("Qual sua maior qualidade? (coragem, astucia, lealdade, sabedoria)");

if (qualidade === "coragem") {
  console.log("Grifinória!");
} else if (qualidade === "astucia") {
  console.log("Sonserina!");
} else if (qualidade === "lealdade") {
  console.log("Lufa-Lufa!");
} else if (qualidade === "sabedoria") {
  console.log("Corvinal!");
} else {
  console.log("Hmm, difícil... muito difícil... Melhor tentar de novo.");
}

// --- 3. Duelo de Bruxos: Usando Operadores Lógicos ---
const temVarinha = true;
const inimigo = "Comensal da Morte";

// Para duelar, você precisa ter uma varinha E o inimigo ser um Comensal
if (temVarinha && inimigo === "Comensal da Morte") {
  console.log("Expelliarmus! Você venceu o duelo.");
} else {
  console.log("Melhor fugir!");
}

4. Desafios do Torneio Tribruxo

Nível: Primeiro Ano (Fácil)

01

Crie uma variável `pontosCasa` com valor 150. Use `if` para verificar se os pontos são maiores que 100 e exiba "Grifinória na liderança!" no console.

02

Crie uma variável `inimigo` com valor "Draco Malfoy". Use `if/else` para verificar se o inimigo é "Voldemort". Se for, exiba "Corra!"; senão, exiba "Duelo!".

03

Crie uma variável `ganhouQuadribol` com valor `true`. Use `if` para verificar se é verdadeira e exibir "Festa no Salão Comunal!".

Nível: N.O.M.s (Médio)

04

Adivinhe o Feitiço: Use `prompt` para perguntar "O que o feitiço 'Wingardium Leviosa' faz?". Se a resposta for "levitar objetos", exiba um alerta de "Correto! +10 pontos para sua casa!". Senão, alerte "Errado!".

05

Poção Polissuco: Use `confirm` para perguntar "A poção está borbulhando?". Se for `true`, use `else if` e `confirm` de novo para perguntar "Tem um fio de cabelo do seu alvo?". Se sim, alerte "Transformação bem-sucedida!". Senão, alerte "Ingrediente faltando!".

Nível: Auror (Desafio)

06

Mapa do Maroto:
1. Use `prompt` para perguntar o nome de uma pessoa em Hogwarts.
2. Use `if`, `else if` e `else` para checar nomes específicos. Se for "Snape", alerte "Ele está na sala de poções.". Se for "Dumbledore", alerte "Ele está na sala do diretor.". Se for "Filch", alerte "Ele está patrulhando o terceiro andar.".
3. Se for qualquer outro nome, use um `else` final para alertar "Não vejo essa pessoa no mapa... Malfeito feito!".

07

Acesso à Plataforma 9 3/4:
1. Use `prompt` para perguntar "Qual o número da plataforma?".
2. Use `confirm` para perguntar "Você é um bruxo?".
3. Use um `if` com o operador `&&` (E) para verificar se a plataforma é "9 3/4" **E** se a pessoa é um bruxo (`true`).
4. Se ambas as condições forem verdadeiras, alerte "Você atravessou a parede! Bem-vindo ao Expresso de Hogwarts!". Caso contrário, alerte "Você bateu na parede! Acesso negado aos trouxas.".
