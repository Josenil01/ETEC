Aula 05: Arrays e Objetos - A Força dos Dados

Organize a Aliança Rebelde (Arrays) e detalhe a Millennium Falcon (Objetos) para dominar as estruturas de dados fundamentais.

1. Uma Galáxia de Informações

Até agora, guardamos informações simples em variáveis. Mas e se precisarmos de uma lista de todos os Jedi do Conselho? Ou da ficha técnica completa de um X-Wing? Para isso, usamos estruturas de dados mais poderosas: **Arrays** para listas ordenadas e **Objetos** para agrupar detalhes sobre uma única coisa.

2. Os Holocrons Jedi: Arrays e Objetos

Array [ ] Uma lista ordenada de itens. Pense em todos os Stormtroopers em uma fileira. Acessamos cada um por sua posição (índice), começando em 0.
Objeto { } Uma coleção de propriedades (chave-valor). Pense na ficha de um dróide: `nome: 'R2-D2'`, `funcao: 'astromecânico'`.
Índice vs. Propriedade Arrays usam um índice numérico (`[0]`) para acessar itens. Objetos usam o nome da propriedade (`.nome`) para acessar valores.

3. Construindo a Estrela da Morte (com Dados)

Este é o Caminho.

Um Jedi não copia o conhecimento, ele o constrói. Digite este código para que a Força (do aprendizado) esteja com você.

aula05_alianca_rebelde.js 
// --- 1. ARRAYS: A Aliança Rebelde ---
const aliancaRebelde = [ "Luke Skywalker", "Leia Organa", "Han Solo" ];
console.log("Esquadrão inicial:", aliancaRebelde);

// Adicionando um novo membro ao FINAL com .push()
aliancaRebelde.push("Chewbacca");
console.log("Reforços chegaram:", aliancaRebelde);

// Adicionando um membro no INÍCIO com .unshift()
aliancaRebelde.unshift("Lando Calrissian");
console.log("Um velho amigo se juntou à causa:", aliancaRebelde);

// Removendo o ÚLTIMO membro com .pop()
aliancaRebelde.pop();
console.log("Chewie foi consertar a Falcon...", aliancaRebelde);

// Removendo o PRIMEIRO membro com .shift()
aliancaRebelde.shift();
console.log("Lando foi cuidar de Bespin...", aliancaRebelde);


// --- 2. OBJETOS: A Ficha Técnica da Millennium Falcon ---
const millenniumFalcon = {
  nome: "Millennium Falcon",
  piloto: "Han Solo",
  "ano de fabricação": 60 // BBY
};

// Adicionando uma nova propriedade
millenniumFalcon.copiloto = "Chewbacca";

// Acessando com colchetes (obrigatório para propriedades com espaço)
console.log("Ano:", millenniumFalcon['ano de fabricação']);

// Verificando se uma propriedade existe com 'in'
const temHyperdrive = 'hyperdrive' in millenniumFalcon; // false
console.log(`A nave possui um hyperdrive? ${temHyperdrive}`);


// --- 3. JUNTANDO A FORÇA: Array de Objetos ---
const droids = [
  { nome: "R2-D2", funcao: "Astromecânico" },
  { nome: "C-3PO", funcao: "Protocolo" }
];

// Alterando uma informação específica
droids[0].funcao = "Herói da Aliança";
console.log("Função do R2-D2 atualizada:", droids[0]);
