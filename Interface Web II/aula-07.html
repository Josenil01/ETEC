Aula 07: Laços de Repetição & DOM

Aprenda a usar `for`, `while` e `forEach` para criar, modificar e limpar elementos HTML dinamicamente na página, automatizando sua Pokédex!

1. Temos que Renderizar Todos!

Imagine que você precisa listar todos os 151 Pokémon originais na tela. Você não vai escrever 151 tags <li> ou <div> à mão no seu HTML, certo? É aqui que os laços de repetição se unem ao DOM (Document Object Model). Na programação web moderna, usamos laços no JavaScript para criar, estilizar e injetar elementos na página de forma automática, construindo interfaces ricas a partir de listas de dados.

⚡ Pausa para o Tailwind CSS

Ao longo desta aula, você notará classes como bg-red-500 ou flex. Isso é o Tailwind CSS! Ele é um framework utilitário que permite estilizar componentes injetando classes pré-definidas diretamente no className do JavaScript. É perfeito para criar e estilizar elementos dinamicamente sem precisar criar dezenas de arquivos CSS separados.

2. As Ferramentas Visuais do Treinador

for (para) Ideal quando você sabe a quantidade exata de elementos que quer criar. Exemplo: gerar 6 espaços vazios (tags <div>) para o seu cinto de Pokébolas.
while (enquanto) Perfeito para remover elementos. Exemplo: enquanto uma caixa (container) possuir elementos filhos (firstChild), continue removendo-os para limpar a tela.
forEach (para cada) A melhor forma de percorrer uma lista de dados (Arrays) e transformá-los em HTML. "Para cada Pokémon na lista, crie um Card e adicione na página."

JS Configuração e forEach

script.js 
const pokedexPreview= document.getElementById('live-pokedex-preview');
const cintoPreview = document.getElementById('live-cinto-preview');
const arenaPreview = document.getElementById('live-arena-preview');
const btnFugir = document.getElementById('btn-fugir');

// --- 1. Laço forEach (Listas Complexas) ---
const pokemonData = [
  { nome: 'Bulbasaur', tipo: 'Planta / Veneno', img: '1', cor: 'bg-green-100 border-green-300' },
  { nome: 'Charmander', tipo: 'Fogo', img: '4', cor: 'bg-red-100 border-red-300' },
  { nome: 'Squirtle', tipo: 'Agua', img: '7', cor: 'bg-blue-100 border-blue-300' },
];

pokemonData.forEach((pkmn) => {
  const card = document.createElement('div');
  card.className = `flex cursor-pointer flex-col items-center rounded-xl border p-4 
    shadow-sm transition-transform hover:-translate-y-1 ${pkmn.cor}`;
  card.innerHTML = `
    <img src="https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/${pkmn.img}.png" 
    alt="${pkmn.nome}" class="h-24 w-24 drop-shadow-md">
    <h3 class="mt-2 text-lg font-bold text-slate-800">${pkmn.nome}</h3>
    <span class="text-xs font-bold uppercase tracking-wider text-slate-600 
    opacity-80">${pkmn.tipo}</span>
  `;
  pokedexPreview.appendChild(card);
});

JS Laço for (Repetições Exatas)

script.js (Adicionar ao final) 
// --- 2. Laço for (Repetições Exatas) ---
for (let i = 1; i <= 6; i++) {
  const pokebola = document.createElement('div');
  pokebola.className = 'relative flex h-12 w-12 cursor-pointer items-center justify-center 
    overflow-hidden rounded-full border-4 border-white bg-red-500 shadow-md transition-transform 
    hover:rotate-12';
  pokebola.innerHTML = `
    <div class="absolute bottom-0 h-1/2 w-full bg-white"></div>
    <div class="absolute h-1 w-full bg-slate-800"></div>
    <div class="z-10 h-4 w-4 rounded-full border-2 border-slate-800 bg-white"></div>
  `;
  cintoPreview.appendChild(pokebola);
}

JS Laço while (Condição de Limpeza)

script.js (Adicionar ao final) 
                                
                                    // --- 3. Laço while (Condição de Limpeza) ---
btnFugir.addEventListener('click', () => {
  while (arenaPreview.firstChild) {
    arenaPreview.removeChild(arenaPreview.firstChild);
  }

  arenaPreview.innerHTML = '<span class="font-bold italic text-slate-500">A arena esta limpa.</span>';
  btnFugir.textContent = 'Voce escapou em seguranca!';
  btnFugir.classList.replace('bg-rose-500', 'bg-slate-400');
  btnFugir.classList.replace('hover:bg-rose-600', 'hover:bg-slate-500');
  btnFugir.disabled = true;
});

4. Desafios de Renderização da Liga

Nível: Treinador Iniciante (Fácil)

01

Fabricando Poções (`for`): Crie um laço for que rode de 1 até 3. Dentro dele, use document.createElement('div') para criar uma poção. Adicione a classe Tailwind className = "w-8 h-8 bg-purple-500 rounded-full" e use o appendChild para inseri-la num container selecionado com getElementById.

02

Lista Rápida (`forEach`): Dado um array ['Pikachu', 'Eevee', 'Snorlax']. Selecione um container vazio. Use o forEach para percorrer essa lista. Para cada nome, crie um <span>, defina o innerHTML como o nome do Pokémon, adicione uma classe className = "font-bold text-blue-600" e faça o appendChild no container.

03

Acordando o Snorlax (`while`): Suponha que no seu HTML existe uma <div id="rota-12"> cheia de elementos filhos "Snorlax" bloqueando o caminho. Selecione essa div no JS e use o padrão do laço while (rota12.firstChild) juntamente com removeChild para limpar a passagem completamente!

Nível: Líder de Ginásio (Médio)

04

Dominando o innerHTML: Você tem um array de objetos: [{nome: "Geodude", level: 12}, {nome: "Onix", level: 14}]. Selecione uma div #ginasio-pedra. Use o forEach. Para cada Pokémon da lista, crie uma div e defina o className como "p-4 border-2 border-stone-500 bg-stone-200 rounded-lg". Em seguida, use o innerHTML com crases para injetar um <h3> com o nome e um <p> com o level. Por fim, acople na tela usando appendChild.

Nível: Mestre Pokémon (Desafio)

05

O PC do Bill (Criar e Destruir): Junte tudo o que aprendeu!

1. Selecione uma div #pc-box e um botão #btn-liberar do HTML.
2. Crie um array de objetos com 3 Pokémon contendo `nome` e `img` (URL).
3. Use forEach para preencher o #pc-box criando divs contendo a imagem e o nome (utilize innerHTML e className), anexando com appendChild.
4. Adicione um evento com addEventListener('click') no botão.
5. Dentro do evento de clique: use um laço while para remover todos os Pokémon (removendo todos os firstChild).
6. Após o laço while terminar de rodar, altere o innerHTML do #pc-box para mostrar a mensagem: "Todos os Pokémon foram soltos na natureza!".
