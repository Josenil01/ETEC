Aula 01: Anatomia da Web

O Esqueleto Digital e a Semântica de Conteúdo

Como funciona um Website?

Para construir qualquer interface na web, utilizamos três tecnologias fundamentais que trabalham juntas. Imagine que estamos a construir uma Casa:

🏗️

HTML

A ESTRUTURA

É a planta e os tijolos. No HTML, definimos onde fica a porta, as janelas e as paredes. Sem ele, não há onde pintar ou colocar móveis.

🎨

CSS

A APARÊNCIA

É a pintura e a decoração. Com o CSS, escolhemos a cor das paredes, o tipo de piso e o tamanho das janelas. Ele torna a estrutura agradável.

⚡

JavaScript

A FUNCIONALIDADE

É a eletricidade e a automação. É o comando que abre o portão automático ou o interruptor que acende a luz. Ele traz vida e interatividade à casa.

"O HTML diz o que o site contém. O CSS diz como ele se parece. O JavaScript diz o que ele faz."

1. Definições Fundamentais de HTML

Linguagem de Marcação

HTML significa HyperText Markup Language. Ele usa "tags" para rotular conteúdos. Não é programação lógica, mas sim uma forma de organizar informação para o computador.

Metadados (O Invisível)

Dentro do <head>, colocamos instruções técnicas. É aqui que dizemos ao Google do que o site trata e garantimos que os acentos apareçam corretamente.

Atributos: O Detalhe

Tags podem ter propriedades extras. Por exemplo, <html lang="pt-pt"> usa o atributo lang para definir que o idioma é o português de Portugal.

Hierarquia e Content

Trabalhamos com "caixas" dentro de "caixas". Um parágrafo pode estar dentro de uma secção, que está dentro do corpo do site. A ordem de abertura e fecho é sagrada.

2. Prática: O Código Estrutural

Analise o código abaixo. Note como as tags criam o "esqueleto" (HTML) da nossa página:

<!-- DOCTYPE: Define que o documento segue o padrão moderno do HTML5 -->
<!DOCTYPE html>

<!-- Atributo global 'lang' define o idioma para navegadores e motores de busca -->
<html lang="pt-pt">

<head>
    <!-- Meta charset: Fundamental para suportar acentos e caracteres especiais -->
    <meta charset="UTF-8">
    
    <!-- Viewport: Garante que o site não fique minúsculo em telemóveis -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- O título que o utilizador vê no separador (aba) do navegador -->
    <title>Aula 01 - Primeiros Passos Estruturais</title>
</head>

<body>
    <!-- HEADER: Secção de cabeçalho (O topo do esqueleto) -->
    <header>
        <h1>Bem-vindo ao Mundo do Código</h1>
        <p>Estamos a aprender a <em>marcar conteúdos</em> com semântica.</p>
    </header>

    <!-- HR: Cria uma linha horizontal para separar visualmente os temas -->
    <hr>

    <!-- MAIN: Contém o conteúdo exclusivo e principal da página -->
    <main>
        
        <!-- SECTION: Agrupa conteúdos relacionados num mesmo bloco -->
        <section>
            <h2>A Importância da Estrutura</h2>
            <p>Um site sem HTML é como um edifício sem <strong>vigas de suporte</strong>.</p>
            
            <!-- BR: Força uma quebra de linha simples no texto -->
            <p>Primeiro organizamos os dados,<br>depois damos-lhes estilo e cor.</p>
        </section>

        <!-- ASIDE: Informação lateral ou nota rápida sobre o assunto -->
        <aside>
            <h3>Sabia que?</h3>
            <p>O primeiro site do mundo era apenas HTML puro!</p>
        </aside>

    </main>

    <hr>

    <!-- FOOTER: O rodapé com créditos e avisos finais -->
    <footer>
        <!-- SMALL: Texto em tamanho reduzido para notas legais -->
        <p><small>&copy; 2026 - Guia de Aprendizagem Web</small></p>
    </footer>

</body>
</html>

3. Atividade de Prática

Para exercitar a sua visão de "Arquitecto Web", modifique o código base:

1

Identidade: Altere o <title> para conter o seu nome e a frase "Meu Portfólio Inicial".

2

Novas Paredes: Crie uma nova <section> no final do <main> sobre as suas "Aspirações Profissionais".

3

Nota Lateral: Adicione uma tag <aside> com um parágrafo sobre qual das analogias (casa ou corpo) mais lhe ajudou a entender o Front-end.

4

Enfâse Visual: No seu novo parágrafo, utilize <strong> e <em> para destacar as competências que deseja adquirir.

5

Finalização: Adicione uma linha <hr> logo abaixo do cabeçalho e outra antes do rodapé para organizar melhor as secções.
