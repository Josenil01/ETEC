Teórica

1) Diferença entre id e class

Explique a diferença entre id e class no HTML e cite um caso de uso para cada.

Resposta

id identifica um elemento único na página (não deve repetir). Exemplo: seção principal id="principal" para link interno.
class agrupa vários elementos com o mesmo estilo/comportamento. Exemplo: vários cartões com class="card-produto".

Teórica

2) Função das tags semânticas

Qual a função das tags main, section, article e aside?

Resposta

main: conteúdo principal e único da página.
section: divisão temática do conteúdo.
article: bloco independente (post/notícia) que faz sentido sozinho.
aside: conteúdo complementar (barra lateral, dica, publicidade).

Teórica

3) Diferença entre métodos GET e POST

Compare GET e POST em formulários HTML.

Resposta

GET: envia dados na URL, ideal para buscas e filtros (dados não sensíveis).
POST: envia dados no corpo da requisição, mais apropriado para cadastro, login e envio de informações sensíveis.

Prática

4) Estrutura básica de página HTML

Criar página com estrutura mínima, título e parágrafo de apresentação.

Resposta

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Minha Primeira Página</title>
</head>
<body>
  <h1>Bem-vindo</h1>
  <p>Esta é minha página introdutória em HTML.</p>
</body>
</html>
Prática

5) Menu com lista não ordenada e links

Montar menu de navegação com ul e 4 links.

Resposta

<nav>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#sobre">Sobre</a></li>
    <li><a href="#servicos">Serviços</a></li>
    <li><a href="#contato">Contato</a></li>
  </ul>
</nav>
Prática

6) Seção com imagem e vídeo

Inserir img com alt e video com controls e poster.

Resposta

<section>
  <h2>Multimídia</h2>
  <img src="imagens/banner.jpg" alt="Banner principal do site" width="420">

  <video controls poster="imagens/capa-video.jpg" width="420">
    <source src="videos/apresentacao.mp4" type="video/mp4">
    Seu navegador não suporta vídeo.
  </video>
</section>
Prática

7) Tabela de notas com thead, tbody e tfoot

Criar tabela com 3 alunos e média no rodapé.

Resposta

<table border="1">
  <thead>
    <tr>
      <th>Aluno</th>
      <th>Nota</th>
    </tr>
  </thead>
  <tbody>
    <tr><td>Ana</td><td>8.5</td></tr>
    <tr><td>Bruno</td><td>7.0</td></tr>
    <tr><td>Carla</td><td>9.0</td></tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Média</td>
      <td>8.17</td>
    </tr>
  </tfoot>
</table>
Prática

8) Formulário de cadastro com validação básica

Campos de nome, email, senha e idade com label, for e required.

Resposta

<form action="#" method="post">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <label for="senha">Senha:</label>
  <input type="password" id="senha" name="senha" required>

  <label for="idade">Idade:</label>
  <input type="number" id="idade" name="idade" min="0" required>

  <button type="submit">Cadastrar</button>
</form>
Prática

9) Rádio e checkbox no mesmo formulário

Escolha única (turno) e múltipla (interesses).

Resposta

<form action="#" method="post">
  <p>Turno:</p>
  <label><input type="radio" name="turno" value="manha" required> Manhã</label>
  <label><input type="radio" name="turno" value="tarde"> Tarde</label>
  <label><input type="radio" name="turno" value="noite"> Noite</label>

  <p>Interesses:</p>
  <label><input type="checkbox" name="interesses" value="html"> HTML</label>
  <label><input type="checkbox" name="interesses" value="css"> CSS</label>
  <label><input type="checkbox" name="interesses" value="js"> JavaScript</label>
</form>
Prática

10) Formulário completo com fieldset, legend, select e textarea

Construir formulário final de matrícula.

Resposta

<form action="#" method="post">
  <fieldset>
    <legend>Matrícula</legend>

    <label for="aluno">Nome do aluno:</label>
    <input type="text" id="aluno" name="aluno" required>

    <label for="curso">Curso:</label>
    <select id="curso" name="curso" required>
      <option value="">Selecione</option>
      <option value="ds">Desenvolvimento de Sistemas</option>
      <option value="adm">Administração</option>
      <option value="log">Logística</option>
    </select>

    <label for="obs">Observações:</label>
    <textarea id="obs" name="obs" rows="4" placeholder="Digite aqui..."></textarea>

    <button type="submit">Enviar</button>
  </fieldset>
</form>
