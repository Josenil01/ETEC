Aula 11: Planos de Fundo e Imagens

Como usar background para construir atmosfera, contraste e leitura sem quebrar a performance.

Cenario da Interface

Se o conteudo e o ator principal, o background e o cenario. Um cenario bem feito destaca o que importa; um cenario mal feito distrai e reduz a legibilidade.

Background-color

Define a base visual do bloco. E a propriedade mais simples para garantir contraste e hierarquia.

Background-image

Aplica imagem de fundo usando url(...). Pode ser combinada com gradientes.

Background-repeat

Controla repeticao: repeat, no-repeat, repeat-x, repeat-y.

Background-position

Move o foco da imagem para pontos como center, top right e valores em %.

Nota didatica: Sempre valide contraste entre texto e fundo. Visual bonito sem legibilidade nao cumpre objetivo de interface.

1. Propriedades Essenciais

Background-size

cover cobre toda a area, contain mostra imagem inteira sem recorte.

Background-attachment

fixed cria efeito de profundidade durante scroll, mas deve ser usado com moderacao.

Shorthand

Voce pode combinar varias propriedades: cor, imagem, repeticao, posicao e tamanho em uma linha.

Camadas

Permite sobrepor gradiente + imagem para controlar contraste sem editar arquivo de imagem.

2. Pratica: Cartao com Fundo em Camadas

O exemplo combina gradiente semitransparente com imagem para manter leitura do texto.

<!DOCTYPE html>
<html lang="pt-PT">
  <head>
    <style>
      .hero-card {
        width: 340px;
        min-height: 220px;
        border-radius: 16px;
        padding: 24px;
        color: #fff;
        background-image:
          linear-gradient(135deg, rgba(15, 118, 110, .82), rgba(13, 148, 136, .45)),
          url('https://images.unsplash.com/photo-1498050108023-c5249f4df085?auto=format&fit=crop&w=1200&q=80');
        background-size: cover;
        background-position: center;
        background-repeat: no-repeat;
      }

      .hero-card h2 { margin: 0 0 8px; }
      .hero-card p { margin: 0; line-height: 1.5; }
    </style>
  </head>
  <body>
    <section class="hero-card">
      <h2>Plano de Fundo com Contraste</h2>
      <p>Imagem + gradiente para destacar informacao sem perder legibilidade.</p>
    </section>
  </body>
</html>

3. Atividade: Diretor de Cenario

  1. Troque a imagem do exemplo por outra e ajuste background-position para manter o foco do objeto principal.
  2. Altere background-size de cover para contain e descreva a diferenca visual.
  3. Adicione background-attachment: fixed; em uma secao alta e observe o efeito no scroll.
  4. Crie uma variacao sem imagem, apenas com linear-gradient, mantendo bom contraste de texto.
  5. Use a forma shorthand de background para escrever as propriedades em uma linha.

4. Quiz Rapido

1. Qual valor evita repeticao da imagem? Resposta: no-repeat.
2. Qual opcao normalmente preenche toda a area, mesmo com recorte? Resposta: background-size: cover.
3. Qual tecnica melhora leitura sobre imagem? Resposta: camada de gradiente semitransparente sobre o fundo.
