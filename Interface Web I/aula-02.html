Portas e Inventários

Na Aula 01, construímos as paredes. Agora, precisamos de portas para circular e armários para organizar.

Links: As Portas

Um site não é uma ilha. Os links (âncoras) são as portas que ligam uma sala à outra ou a sua casa ao resto do mundo (a Internet). O atributo href é o endereço para onde a porta leva.

Listas: Os Armários

Informação sem ordem é ruído. As listas permitem-nos criar inventários, menus e manuais de instrução. Elas agrupam itens relacionados, facilitando a "leitura rápida" do utilizador.

1. Conceitos Fundamentais

Âncoras (<a>)

O motor da Web. Usa o atributo href (referência) para navegar. Pode abrir na mesma aba ou numa nova com target="_blank".

Caminhos (Paths)

Podem ser Absolutos (URL completa de outro site) ou Relativos (diretórios dentro do seu próprio projeto).

Listas não Ordenadas

A tag <ul> (Unordered List) cria listas com marcadores (bolinhas). Ideal para menus ou itens sem hierarquia cronológica.

Listas Ordenadas

A tag <ol> (Ordered List) gera números ou letras automaticamente. Perfeita para passos de uma receita ou rankings.

2. Prática Explicada

Este código simula uma página de recursos. Analise cada linha para entender a conexão entre as tags:

<!DOCTYPE html>
<html lang="pt-pt">
<head>
    <meta charset="UTF-8">
    <title>Aula 02 - Navegação Interativa</title>
</head>
<body>

    <!-- O NAV é a área semântica ideal para menus de navegação -->
    <nav>
        <h2>Menu de Acesso Rápido</h2>
        <!-- Criamos um menu usando uma lista para melhor organização -->
        <ul>
            <!-- O href="#" cria um link que aponta para o topo da própria página -->
            <li><a href="#">Início</a></li>
            
            <!-- Link relativo: aponta para um ficheiro na mesma pasta -->
            <li><a href="contato.html">Contacto Local</a></li>
            
            <!-- Link absoluto: aponta para um endereço externo completo -->
            <li><a href="https://google.com" target="_blank">Google (Nova Aba)</a></li>
        </ul>
    </nav>

    <hr>

    <main>
        <section>
            <h3>Como Aprender Desenvolvimento Web em 3 Passos:</h3>
            <!-- Listas ordenadas (ol) geram a numeração 1, 2, 3 automaticamente -->
            <ol>
                <li>Entenda a estrutura básica (HTML)</li>
                <li>Aplique estilos visuais (CSS)</li>
                <li>Crie interatividade dinâmica (JavaScript)</li>
            </ol>
        </section>

        <section>
            <h3>Ferramentas Recomendadas:</h3>
            <!-- Listas não ordenadas (ul) usam marcadores circulares -->
            <ul>
                <li>Editor de Código (VS Code)</li>
                <li>Navegador Moderno (Chrome/Firefox)</li>
                <li>Documentação da MDN Web Docs</li>
            </ul>
        </section>
    </main>

    <footer>
        <p>Deseja falar connosco? <a href="mailto:ajuda@exemplo.com">Envie um E-mail</a></p>
    </footer>

</body>
</html>

3. Atividade: Prática de Navegação

Aplique as seguintes modificações no código de exemplo para fixar o conteúdo:

1

No menu de navegação, adicione um novo item de lista que leve para o seu perfil do GitHub ou LinkedIn.

2

Mude a lista de "Como Aprender" de Ordenada (<ol>) para Não Ordenada (<ul>) e veja a diferença visual.

3

Crie um link no rodapé que permita ao utilizador ligar diretamente para um número fictício usando o prefixo tel: no href.

4

Dentro da secção de "Ferramentas", adicione uma sub-lista (uma lista dentro de outra) com versões específicas dos softwares citados.

5

Adicione o atributo title="Clique para ir ao topo" no primeiro link do menu e teste o resultado ao passar o rato (mouse) sobre ele.
