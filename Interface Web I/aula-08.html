Aula 08: Cores e Voz Visual

Dominando a Psicologia Cromática e a Tipografia Profissional

O Clima e a Voz

Imagine que o seu site é uma Sala de Estar.

A Cor é o Clima

As cores definem a temperatura da sala. Um azul suave traz calma e confiança (ideal para bancos); um laranja vibrante traz fome e urgência (ideal para comida rápida). A cor dita como o utilizador se deve sentir.

A Fonte é a Voz

A tipografia é o tom de voz de quem está na sala. Fontes com serifa (pequenos traços nas pontas) falam com autoridade e tradição. Fontes sem serifa falam com modernidade e clareza.

1. Sistemas de Cores e Tipos

HEX e RGB

HEX: Código de 6 dígitos (#FFFFFF). RGB: Mistura de Vermelho, Verde e Azul (rgb(255,255,255)). São os "DNAS" das cores no monitor.

Serif vs Sans-Serif

Serif: Fontes com "pés" (ex: Times). Transmitem seriedade. Sans-Serif: Fontes retas (ex: Arial). Transmitem limpeza e tecnologia.

Line-height e Spacing

O segredo do design profissional está no respiro. O line-height controla a distância entre linhas, e o letter-spacing a distância entre letras.

Web Fonts

Não estamos limitados às fontes do sistema. Usamos o Google Fonts para carregar tipografias externas que tornam o site único.

2. Prática: Estilo Tipográfico Avançado

Observe como este código aplica cores em diferentes formatos e utiliza uma tipografia externa para criar contraste:

<!DOCTYPE html>
<html lang="pt-pt">
<head>
    <meta charset="UTF-8">
    <!-- Importando fontes do Google para usar no CSS -->
    <link href="https://fonts.googleapis.com/css2?family=Playfair+Display&family=Roboto&display=swap" rel="stylesheet">
    
    <style>
        body {
            /* Cor de fundo usando sistema HEX (Cinza muito claro) */
            background-color: #fdfdfd;
            /* Cor do texto principal usando RGB (Preto suave) */
            color: rgb(40, 40, 40);
            /* Definindo a fonte Roboto (Sans-Serif) como padrão */
            font-family: 'Roboto', sans-serif;
            font-size: 18px;
            /* Espaçamento entre linhas para melhorar a leitura */
            line-height: 1.8;
        }

        h1 {
            /* Cor usando nome da cor (Sienna) */
            color: sienna;
            /* Fonte Playfair (Serifada) para títulos elegantes */
            font-family: 'Playfair Display', serif;
            font-size: 48px;
            margin-bottom: 10px;
            /* Espaçamento entre letras para um aspeto de revista */
            letter-spacing: -1px;
        }

        .destaque-cor {
            /* Cor de fundo usando HEX com opacidade (transparência) */
            background-color: #ffd70055; /* Amarelo Ouro com 55 de transparência */
            padding: 5px 10px;
            font-weight: 700; /* Peso da fonte: Negrito */
        }

        footer {
            text-align: center;
            font-size: 14px;
            color: #888;
            /* Transformação: deixa tudo em maiúsculas */
            text-transform: uppercase;
            margin-top: 50px;
        }
    </style>
</head>
<body>

    <header>
        <h1>A Arte da Leitura</h1>
        <p>Design não é apenas o que se vê, é como se <span class="destaque-cor">percebe</span>.</p>
    </header>

    <main>
        <p>
            Quando escolhemos as cores certas, guiamos o olhar do utilizador. 
            A tipografia correta garante que a mensagem chegue sem esforço.
        </p>
    </main>

    <footer>
        Direitos de Design Reservados &copy; 2026
    </footer>

</body>
</html>

3. Atividade: O Designer de Cores

Aplique estas melhorias visuais para testar os seus conhecimentos:

1

Cores HEX: Mude a cor do h1 para #2c3e50 (um azul petróleo profundo).

2

Legibilidade: Altere o line-height do body para 2.0. Note como o texto ganha ainda mais "respiro".

3

Contraste: Mude a cor da classe .destaque-cor para um fundo preto (#000) e a cor do texto para branco (#fff).

4

Peso Visual: Adicione a propriedade font-weight: 300; ao rodapé para deixá-lo com uma aparência mais fina e discreta.

5

Personalização: Visite o Google Fonts, escolha uma fonte diferente e tente substituir o 'Roboto' no seu código (apenas no atributo font-family).
