Aula 09: A Anatomia da Caixa

Dominando o Conteúdo, o Enchimento e a Borda

A Encomenda Postal

No CSS, tudo é uma caixa. Imagine que está a enviar um Objeto Frágil pelo correio:

O Produto (Content)

É o objeto em si (o texto ou a imagem). A sua largura (width) e altura (height) definem o espaço básico que ele ocupa.

Plástico Bolha (Padding)

É o espaço interno. Serve para proteger o produto, impedindo que ele encoste nas paredes da caixa. Dá "respiro" ao conteúdo.

A Caixa (Border)

É a barreira física que delimita o objeto. Pode ser fina, grossa, colorida ou invisível, mas ela está sempre lá definindo o limite.

Distância (Margin)

É o espaço externo entre a sua caixa e as outras caixas no camião de entregas. (Exploraremos mais na Aula 10).

1. Propriedades de Estrutura

Padding (Espaço Interno)

Empurra o conteúdo para dentro. Pode ser definido para todos os lados ou individualmente: padding-top, padding-right, etc.

Border (A Moldura)

Precisa de três valores: Largura (2px), Estilo (solid) e Cor (black). Ex: border: 2px solid #000;.

Width e Height

Definem o tamanho da área de conteúdo. Por padrão, o padding e a borda são somados a estes valores, aumentando o tamanho total da caixa.

Box-sizing: border-box

O "salvador" dos layouts modernos. Faz com que o padding e a borda sejam incluídos dentro da largura definida, não somados por fora.

2. Prática: Construindo a Caixa Perfeita

Este exemplo demonstra como o Padding e a Border alteram visualmente o elemento. Leia os comentários atentamente:

<!DOCTYPE html>
<html lang="pt-pt">
<head>
    <style>
        /* Aplicamos o border-box em tudo para facilitar o cálculo do tamanho */
        * {
            box-sizing: border-box;
        }

        .caixa-exemplo {
            /* O 'PRODUTO': Largura fixa de 400 pixels */
            width: 400px;
            background-color: #3498db; /* Azul vibrante */
            color: white;
            
            /* O 'PLÁSTICO BOLHA': 40px de respiro interno em todos os lados */
            padding: 40px;
            
            /* A 'CAIXA': Uma borda sólida, escura e de 10 pixels de espessura */
            border: 10px solid #2c3e50;

            /* Alinhamento de texto dentro da caixa */
            text-align: center;
            font-size: 20px;
        }

        .caixa-variada {
            width: 400px;
            margin-top: 20px;
            background-color: #ecf0f1;
            color: #333;
            border: 2px dashed #7f8c8d; /* Borda tracejada */

            /* PADDINGS INDIVIDUAIS: Topo, Direita, Baixo, Esquerda */
            padding-top: 10px;
            padding-bottom: 50px;
            padding-left: 20px;
        }
    </style>
</head>
<body>

    <div class="caixa-exemplo">
        Eu sou uma caixa azul com muito preenchimento interno!
    </div>

    <div class="caixa-variada">
        Esta caixa tem espaços internos diferentes em cada lado.
    </div>

</body>
</html>

3. Atividade: O Arquiteto de Caixas

Realize as seguintes modificações no seu ficheiro para dominar o Box Model:

1

Respiro Extra: Na .caixa-exemplo, aumente o padding para 80px. Observe como o texto se afasta da borda.

2

Estilo de Borda: Mude o solid da primeira caixa para double ou dotted. Qual é o efeito visual?

3

Cálculo de Tamanho: Desative a linha box-sizing: border-box; comentando-a com /* ... */. Note como a caixa azul parece "esticar" para fora.

4

Desafio de Lados: Na .caixa-variada, adicione um padding-right: 100px; e veja o que acontece com o texto quando ele chega perto do lado direito.

5

Arredondamento: Adicione border-radius: 20px; à caixa azul. Lembre-se: a borda é a "parede" da caixa!
