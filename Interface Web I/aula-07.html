Aula 07: O Poder do Estilo

Introdução ao CSS, Formas de Inclusão e Seletores

A Maquilhagem da Web

Se o HTML é o esqueleto, o CSS é a pele, a roupa e a maquilhagem. É o que transforma dados brutos numa experiência visual agradável.

Cascata: A Ordem de Comando

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). Isso significa que as regras "escorrem" de cima para baixo. Se houver dois comandos diferentes para a mesma coisa, o que está mais abaixo ou é mais específico costuma ganhar.

Seletores: O Alvo

Construir CSS é como dar ordens a um pintor: "Pinte as paredes (Tipo), mas apenas as paredes da sala (Classe) e, especificamente, aquela parede de destaque (ID)".

1. Formas de Inclusão e Seletores

Inline, Interno e Externo

Podemos colocar CSS diretamente na tag (Inline), numa tag <style> (Interno) ou num ficheiro .css separado (Externo - O método profissional).

Seletor de Tipo

Alvo: Todas as tags de um tipo. Ex: h1 { color: blue; } afeta todos os H1 do site.

Seletor de Classe (.)

Alvo: Qualquer elemento com a classe. Usa-se um ponto antes do nome. Ex: .botao-verde { background: green; }.

Seletor de ID (#)

Alvo: Um elemento único. Usa-se um cardinal (hashtag). Ex: #logotipo { width: 100px; }. Tem prioridade máxima sobre classes.

2. Prática: Estilizando a Primeira Página

Observe como o código abaixo utiliza os três tipos de seletores básicos e o método de CSS Interno:

<!DOCTYPE html>
<html lang="pt-pt">
<head>
    <meta charset="UTF-8">
    <title>Aula 07 - Meu Primeiro CSS</title>

    <!-- CSS INTERNO: Estilos definidos dentro da tag style no head -->
    <style>
        /* SELETOR DE TIPO: Afeta todas as tags 'body' */
        body {
            background-color: #f0f0f0; /* Define a cor de fundo cinza claro */
            font-family: Arial, sans-serif; /* Define a fonte padrão */
        }

        /* SELETOR DE TIPO: Afeta todos os títulos 'h1' */
        h1 {
            color: navy; /* Cor azul marinho */
            text-align: center; /* Centraliza o texto */
        }

        /* SELETOR DE CLASSE: Afeta qualquer tag com class="card" */
        .card {
            background-color: white;
            padding: 20px;
            border-radius: 10px;
            margin: 20px auto;
            width: 300px;
            box-shadow: 0 4px 8px rgba(0,0,0,0.1);
        }

        /* SELETOR DE ID: Afeta apenas o elemento único com id="destaque" */
        #destaque {
            color: crimson;
            font-weight: bold;
            text-decoration: underline;
        }
    </style>
</head>
<body>

    <h1>Iniciando no Mundo Visual</h1>

    <!-- Elemento que recebe a classe 'card' -->
    <div class="card">
        <p>Este é um cartão estilizado com classe.</p>
        <!-- Elemento com ID único -->
        <p id="destaque">Eu sou um texto de destaque único!</p>
    </div>

    <!-- CSS INLINE: Estilo direto na tag (usar com moderação!) -->
    <p style="text-align: center; color: gray;">
        Texto centralizado via CSS Inline.
    </p>

</body>
</html>

3. Atividade: O Estilista de Código

Desafie-se a modificar o estilo da página anterior:

1

Mudança de Fundo: No seletor body, altere a cor de fundo para #e0f7fa (um azul piscina claro).

2

Estilo de Título: No seletor h1, adicione a propriedade text-transform: uppercase; para deixar o título em maiúsculas.

3

Personalização do Card: No seletor .card, mude o border-radius para 0px e adicione uma borda: border: 2px solid navy;.

4

Prioridade de ID: Crie uma classe chamada .texto-azul que define color: blue;. Aplique esta classe ao parágrafo que já tem o id="destaque". Qual cor venceu? (Isso prova a prioridade do ID).

5

Novo Seletor: Adicione um seletor para a tag p (tipo) definindo line-height: 1.6; (espaçamento entre linhas) e veja como a leitura melhora.
