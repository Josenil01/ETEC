2. A Peneira: WHERE e os Operadores

Para aplicar a Seleção (corte horizontal), utilizamos a cláusula WHERE logo após o FROM. O banco de dados analisa a sua condição e pergunta linha a linha: "Isto é Verdade ou Mentira?". Só as linhas "Verdadeiras" aparecem no resultado.

SELECT nome, salario
FROM funcionario
WHERE salario > 4000;

Tradução: "Traga-me os nomes e salários da tabela de funcionários, MAS APENAS se o salário for estritamente maior que 4000."

O Arsenal de Comparação

Operador Significado Exemplo Prático

O Paradoxo do Alias: Ordem de Execução

Muitos iniciantes tentam filtrar uma coluna virtual criada no SELECT (ex: WHERE salario_anual > 50000). O banco dará ERRO. Porquê? Porque o banco lê a consulta nesta ordem:

1º FROM
2º WHERE
3º SELECT

Como o WHERE acontece antes do SELECT, o banco ainda não sabe que o "apelido" (Alias) existe! A solução? Faça o cálculo matemático diretamente no WHERE. O próprio filtro é uma calculadora.
