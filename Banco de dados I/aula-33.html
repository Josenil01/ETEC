Modulo 6: DQL Avancado Operacoes de Conjunto

Aula 33: O Arsenal de Conjuntos

UNION, UNION ALL, INTERSECT e EXCEPT para combinar, cruzar e diferenciar resultados de consultas.

1. Ideia Central

Operacoes de conjunto permitem tratar resultados SQL como conjuntos matematicos. Em vez de apenas filtrar linhas, voce passa a unir, intersectar e subtrair resultados de duas consultas.

2. Regras de Ouro

Compatibilidade

As consultas devem retornar o mesmo numero de colunas e tipos compativeis.

Ordenacao

Use ORDER BY apenas no final do conjunto completo.

3. Laboratorio SQL

script-union.sql 
-- UNION remove duplicados
SELECT email FROM clientes_filial_a
UNION
SELECT email FROM clientes_filial_b
ORDER BY email;
script-intersect-except.sql 
-- INTERSECT: clientes em ambas as filiais
SELECT email FROM clientes_filial_a
INTERSECT
SELECT email FROM clientes_filial_b;

-- EXCEPT: clientes apenas da filial A
SELECT email FROM clientes_filial_a
EXCEPT
SELECT email FROM clientes_filial_b;

4. Pratica no Neon

Fluxo recomendado: copie os scripts, abra o Neon e execute no SQL Editor.
