Aula 28: A Revolução das Dimensões

Agrupamento de Dados: Aprenda a organizar informações em categorias e a filtrar resultados consolidados com GROUP BY e HAVING.

1. O Conceito dos Baldes (Buckets)

A cláusula GROUP BY funciona como uma triagem. Ela separa as linhas em "baldes" baseados numa coluna comum. O banco calcula a função agregada (SUM, AVG, COUNT) para cada balde individualmente.

Balde TI (3 funcionários)
Balde RH (2 funcionários)
GROUP BY id_depto

O resultado será uma linha por departamento.

2. A Regra Sagrada

Se usar uma função agregada (como COUNT), todas as outras colunas no SELECT têm de estar no GROUP BY.

ERRO

SELECT nome, COUNT(*) FROM func GROUP BY id_depto;

O banco não sabe qual "nome" mostrar para o grupo do departamento.

CERTO

SELECT id_depto, COUNT(*) FROM func GROUP BY id_depto;

1

Ticket #101: RH (Censo)

"Arquiteto, precisamos mapear a distribuição de pessoal. Liste cada id_depto e quantos funcionários estão alocados em cada um."

SELECT id_depto, COUNT(*) FROM funcionario GROUP BY id_depto;
2

Ticket #102: Financeiro (Custos)

"Para o relatório de custos, projete o id_depto, a soma (apelidada de 'total') e a média ('media') salarial agrupados por setor."

SELECT id_depto, SUM(salario) AS total, AVG(salario) AS media FROM funcionario GROUP BY id_depto;
3

Ticket #103: RH (Especialistas)

"Precisamos saber quantas pessoas temos em cada profissão. Agrupe pelo cargo e conte o total em cada categoria."

SELECT cargo, COUNT(*) FROM funcionario GROUP BY cargo;
4

Ticket #104: Diretoria (Densidade)

"Só nos interessam os setores com equipas grandes. Mostre os id_depto e a contagem total, mas filtre para exibir apenas departamentos que tenham mais de 2 funcionários."

SELECT id_depto, COUNT(*) FROM funcionario GROUP BY id_depto HAVING COUNT(*) > 2;
5

Ticket #105: CFO (Relatório Elite)

"Atenção ao Pipeline! Calcule a média salarial por id_depto, mas considere apenas funcionários que ganham mais de 3000. No final, exiba apenas os departamentos cuja média resultante seja superior a 4000."

SELECT id_depto, AVG(salario) FROM funcionario WHERE salario > 3000 GROUP BY id_depto HAVING AVG(salario) > 4000;
6

Ticket #106: Debug (O Erro do Estagiário)

"O banco está a dar erro na query abaixo. Conserte-a para que ela mostre apenas o id_depto e o total de funcionários por setor, respeitando a Regra Sagrada."

SELECT id_depto, COUNT(*) FROM funcionario GROUP BY id_depto;
